# Example 3: Review ДДУ перед подписанием (214-ФЗ)

## Контекст

Физ.лицо покупает квартиру в новостройке через ДДУ (Договор Долевого Участия). Цена 8 млн ₽, ипотека 70%. Юрист (или сам покупатель если разбирается) review draft ДДУ от застройщика перед подписанием.

**Stakes:** срок строительства типично 18-24 мес; при проблемах застройщика — риск потери всей суммы.

## Skill

`real-estate/ddu-review` — killer skill для 214-ФЗ с 8 проверочными секциями.

## User prompt

```
Юрист: /ddu-review

Объект: 2-комнатная квартира, 65 м², 7 этаж
ЖК "Зелёные просторы", корпус 3
Москва, ЮЗАО
Цена: 8,000,000 ₽
Эскроу: указан Сбербанк
Срок передачи: "до 31.12.2027"
Гарантия: указано 5 лет

Застройщик: ООО "СпецСтрой Капитал"
ИНН: 7707009999

[прикреплен draft ДДУ на 25 страниц]
```

## Flow

```
LLM читает SKILL.md (~750 строк)
↓
Tool call 1: egrul.find_company(inn="7707009999")
  → {name: "ООО СпецСтрой Капитал", founded: "2018",
     status: "active", director: "Сидоров С.С.",
     capital: 10_000_000, ОКВЭД: "41.20"}
↓
Tool call 2: efrsb.check_company_bankruptcy(inn="7707009999")
  → {status: "no_bankruptcy"}
↓
Tool call 3: kad.search_cases(defendant_inn="7707009999")
  → [12 cases as defendant — большинство о просрочке передачи объектов;
     8 проиграли, 4 в процессе; средняя сумма иска 800k]
↓
Tool call 4: pravo.search_npa(query="214-ФЗ дольщик")
  → confirm current редакция
↓
LLM применяет 8-секционный checklist + risk assessment
```

## Output (выдержка)

```markdown
# ДДУ Review — ООО "СпецСтрой Капитал" / квартира 65м²

## Bottom line

**🟡 PROCEED WITH CHANGES.** Договор в основном compliant с 214-ФЗ, но
**12 проигранных дел о просрочке передачи** — серьёзный red flag. Рекомендую:
1. Negotiate более конкретный срок передачи
2. Усилить неустойку (или просто rely на 1/150 ст.рефинансирования)
3. Дополнительный due diligence на финансовое состояние

## Застройщик assessment

| Параметр | Status | Комментарий |
|----------|--------|-------------|
| ЕГРЮЛ active | ✅ | С 2018 года |
| Не в реестре проблемных | ✅ | Verify на наш.дом.рф |
| Не в банкротстве | ✅ | ЕФРСБ clean |
| Налог.задолженность | ⚠ | Запросить справку ФНС |
| **История судебных проигрышей** | 🔴 | **12 дел defendant; 8 проигрышей** |
| Уставный капитал | ✅ | 10М — нормально для девелопера |

→ **Verdict:** не катастрофа, но **высокий risk просрочки**.

## Compliance с 214-ФЗ — checklist

### Mandatory elements (ст.4 214-ФЗ)

- [✅] Конкретный объект описан (адрес, площадь, этаж)
- [⚠] **Срок передачи "до 31.12.2027"** — vague формулировка
  - Лучше: "не позднее 31.12.2027" с привязкой к разрешению на ввод
  - Risk: застройщик может затянуть до самой даты
- [✅] Цена 8М указана конкретно
- [✅] Гарантия 5 лет соответствует requirement
- [✅] Эскроу Сбербанк указан как способ обеспечения

### Объект (Section A)

- [✅] Кадастровый номер объекта строительства указан
- [✅] Проектная декларация: можно найти на наш.дом.рф (проверить!)
- [✅] Площадь: проектная 65м² ± 5% adjustment clause присутствует
- [⚠] **Парковочное место — отдельный ДДУ** (отдельные 350k ₽) — норм, но fyi

### Цена и оплата (Section C)

- [✅] **Эскроу в Сбербанке** — обязательно с 2019 ✓
- [✅] Условия возврата при non-completion прописаны
- [✅] Single payment в начале — стандарт для ипотеки
- [✅] Госпошлина за регистрацию ДДУ на покупателя (стандарт ~2000 ₽)

### Гарантия (Section D)

- [✅] 5 лет на объект — minimum по 214-ФЗ
- [✅] 3 года на технологическое оборудование
- [⚠] **"Гарантия не распространяется на нормальный износ"** — стандарт, но
  watch против broad interpretation
- [⚠] Условие "за исключением случаев когда работы выполнены сторонней
  организацией по выбору покупателя" — too broad, recommended remove

### Неустойка (Section E)

- [✅] **1/150 ставки рефинансирования ЦБ** на каждый день просрочки —
  это imperative ст.6 214-ФЗ, не может быть reduced

→ В **рублях для нашего случая:**
  Если просрочка 6 мес от 31.12.2027 (т.е. до 30.06.2028):
  - 8,000,000 × (16% / 150) × 182 дня = ~155,000 ₽ неустойка
  - Plus моральный вред (1-50k) — суд назначит
  - Plus судебные расходы

Это **встроенная защита** — задержать застройщика финансово.

### Расторжение (Section G)

- [✅] **Право unilateral расторжения при просрочке > 2 мес** — присутствует
  как требует ст.9 214-ФЗ
- [⚠] "Только через суд" — formulация ущемляющая, ст.9 даёт unilateral right
- [✅] Возврат уплаченных средств + проценты + неустойка

### Регистрация (Section H)

- [✅] Регистрация в ЕГРН обязательна — указано
- [✅] Эскроу разблокируется **после регистрации** — правильно (защита покупателя)

## Ущемляющие clauses identified

| # | Пункт | Severity | Recommendation |
|---|-------|----------|----------------|
| 1 | "Споры подсудны суду по месту нахождения застройщика" | 🟡 | Negotiate removal или просто rely на ст.17 ЗЗПП — потребитель может выбирать суд |
| 2 | "Расторжение только через суд" | 🟡 | Negotiate fix — ст.9 даёт unilateral; иначе rely на закон |
| 3 | "Запрет уступать права без consent застройщика" | 🟡 | Ст.11 — нельзя ограничить полностью; negotiate condition |
| 4 | "Гарантия не на дополнительную отделку" | 🟢 | Стандарт если отделка не в scope |

## Final recommendations

### Перед подписанием

- [ ] **Verify проектная декларация на наш.дом.рф** — match с ДДУ?
- [ ] **Запросить разрешение на строительство** + ввод (если ещё не получено — flag!)
- [ ] **Финансовая отчётность застройщика** опубликована? Какие debt levels?
- [ ] **Bank check** — Сбербанк подтверждает условия эскроу?
- [ ] **Sequence:** подписать → зарегистрировать в ЕГРН → только потом deposit на эскроу

### Изменения в текст (negotiate)

1. Срок передачи: "не позднее 31.12.2027" вместо "до 31.12.2027"
2. Убрать "споры по месту нахождения застройщика" (или know что can override per ст.17 ЗЗПП)
3. Убрать "расторжение только через суд"

### Если застройщик отказывается изменять

- Стандартная practice — застройщики обычно НЕ соглашаются на крупные изменения
- **Подписывать можно** — ущемляющие clauses в основном overridable законом
- Но **strong recommendation:** не повышать сумму до verify финансовой устойчивости

## Risk если просрочка передачи

- **Probability:** **medium-high** (учитывая 12 проигранных дел)
- **Mitigation:**
  - Через 2 мес после срока — formal претензия о расторжении ст.9
  - Возврат через эскроу — automatic если расторгнуто корректно
  - Иск о неустойке если хочешь компенсацию (но это +12-18 мес)

## Ипотечная специфика

Для ипотечной покупки 70%:
- Банк-кредитор будет independently review ДДУ
- Bank checks застройщика отдельно — additional layer защиты
- Title insurance — recommend на 3 года (~10-30k ₽)

## Disclaimer

ДДУ-сделки на 8М ₽ — не место для AI-only review. **Engage outside-юриста**
со специализацией недвижимость для финального ОК и переговоров с застройщиком.
```

## Что покупатель делает дальше

1. **Не подписывает сразу** — берёт recommended изменения к застройщику
2. **Verify** наш.дом.рф проектную декларацию + разрешение на строительство
3. **Bank** независимо проверяет — берёт их feedback
4. **Outside-юрист** (1-2 часа консультация ~10-30k ₽) — финальный ОК
5. **Подписывает** в нотариусе или офисе застройщика
6. **Регистрация в ЕГРН** через нотариуса / банк / самостоятельно (5-7 раб.дней)
7. **Deposit на эскроу** — только после регистрации

## ROI

- **Manual review** опытным юристом: 3-5 часов × 5000 ₽/час = 15-25k ₽
- **Self-review** через AI: ~10 минут + ~30 min discuss with юрист = **в 5-10 раз быстрее**
- **Risk избежания**: 8М квартиры — спасти даже 200k ₽ на неустойке = ROI огромный
