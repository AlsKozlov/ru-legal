# Example 2: Анализ увольнения по сокращению (ст.81 п.2 ТК)

## Контекст

In-house HR-юрист в IT-компании. Руководство решило сократить 5 разработчиков в связи с реструктуризацией. Нужен анализ рисков и пошаговый план чтобы НЕ проиграть в суде (а суды в РФ pro-работника в 65-80% случаев).

**Stakes:** проигрыш в суде = восстановление + СЗП за вынужденный прогул × 5 работников = ~10М ₽ exposure.

## Skill

`labor-law/termination-review` — killer skill с 13 grounds, матрицей защищённых категорий, 12-step checklist для ст.81 п.2.

## User prompt

```
HR: /termination-review

Ground: ст.81 п.2 ТК (сокращение штата)
Affected: 5 разработчиков middle-уровня
Category breakdown:
  - 2 М 30-35 лет (не в защ.категориях)
  - 1 Ж 28 лет (не беременна, без детей)
  - 1 Ж 34 лет с ребёнком 5 лет (одинокая мать)
  - 1 М 52 года (пенсионером ещё не стал)

Профсоюз в компании: нет
Цель: сокращение штата для оптимизации
```

## Flow

```
LLM читает SKILL.md (~750 строк)
↓
Применяет защ.категории matrix:
  - Иван Иванов 32 — OK для увольнения
  - Пётр Петров 30 — OK
  - Мария Сидорова 28 — OK
  - Анна Кузнецова 34 + одинокая мать ребёнка 5 лет
    → 🔴 ст.261 ч.4 ТК — НЕЛЬЗЯ уволить по сокращению
  - Сергей Орлов 52 — pre-retirement (5 лет до пенсии)
    → 🟡 особый статус, повышенная защита
↓
Tool call (опционально): pravo.search_npa(query="ст.81 ТК")
  → подтверждает текущую редакцию
↓
Генерирует 12-step checklist + risk assessment
```

## Output (выдержка)

```markdown
# Termination Review — сокращение штата 5 работников

## Bottom line

**🔴 STOP.** Текущий список включает 2 защищённые категории:

1. **Анна Кузнецова, 34 — одинокая мать ребёнка 5 лет**
   - **Запрет ст.261 ч.4 ТК** — нельзя уволить по сокращению пока ребёнку < 14 лет
   - Исключения: ликвидация компании / по виновным основаниям (ст.81 п.5-11)
   - **Action:** исключить из списка сокращения

2. **Сергей Орлов, 52 — pre-retirement**
   - 5 лет до общеустановленного пенсионного возраста (60 м/55 ж + 5 лет переходного периода)
   - **Не запрет**, но повышенная защита: УК ст.144.1 (необоснованный отказ в приёме / увольнение работников предпенсионного возраста — штраф до 200k, обязательные работы)
   - **Risk:** ФНС может запросить пояснения почему именно он;
     прокуратура / ГИТ могут initiate проверку
   - **Action:** убедиться что НЕ дискриминация по возрасту;
     документировать business reasons (по skill / performance)

## Безопасно увольнять (3 работника)

- Иван Иванов, Пётр Петров, Мария Сидорова

## 12-step checklist для каждого из 3 работников

### Шаг 1. Решение работодателя о сокращении

- [ ] Приказ о сокращении штата
- [ ] Новое штатное расписание (исключаемые позиции явно отмечены)
- [ ] **Business reasoning** документирован (для защиты в суде):
  - реорганизация структуры
  - технологические изменения
  - финансовое обоснование
- [ ] Дата вступления в силу нового штатного — **не ранее 2 мес** после уведомления работников

### Шаг 2. Преимущественное право на оставление (ст.179)

При сокращении нескольких работников **той же должности** — преимущество остаются:
- Более высокая производительность труда / квалификация
- При равенстве: семейные с двумя+ иждивенцами; единственные кормильцы; получившие проф.заболевание у данного работодателя; и т.д.

→ Для 3 работников разной квалификации — обычно не critical. Но **document the analysis** (зафиксировать что compared производительность).

### Шаг 3. Уведомление работника о сокращении

- [ ] Письменное уведомление **под подпись**
- [ ] **Не менее 2 месяцев** до увольнения
- [ ] Текст: "В связи с сокращением штата... уведомляем... увольнение _DD.MM.YYYY_..."

### Шаг 4. Предложение другой работы (ст.81 ч.3)

- [ ] **Письменно предложить** все имеющиеся вакансии (включая нижестоящие / нижеоплачиваемые) которые работник может выполнять с учётом квалификации / здоровья
- [ ] **Предлагать на протяжении всех 2 месяцев** — при появлении новых вакансий
- [ ] **Документировать** ответ работника на каждое предложение
- [ ] Если нет вакансий — formal note об отсутствии

### Шаг 5. Учёт мнения профсоюза (ст.82)

**В нашем случае профсоюз отсутствует** → step skipped.

### Шаг 6. Не уволить в период болезни / отпуска (ст.81 ч.6)

- [ ] Day-of увольнения — работник не на больничном
- [ ] Не в ежегодном / БСС / учебном отпуске
- [ ] Если на больничном за день до — отложить до выхода

### Шаг 7. Приказ об увольнении

- [ ] Издан **в день увольнения**
- [ ] Основание: "ст.81 п.2 ТК — сокращение численности или штата работников организации"
- [ ] Reference to приказа о сокращении штата

### Шаг 8. Ознакомление с приказом

- [ ] Под подпись **в день издания**
- [ ] Если отказ — акт об отказе

### Шаг 9. Финальные расчёты (ст.140, 178)

В день увольнения работник получает:

1. **Зарплата** за фактически отработанное
2. **Компенсация** за неиспользованный отпуск
3. **Выходное пособие — 1 средний месячный заработок** (ст.178)
4. **2-3 средние зарплаты** в качестве additional пособия если в течение 2 мес после увольнения не трудоустроится (по постановлению службы занятости)

→ **Бюджет на одного работника:** ~3-5 окладов в зависимости от того сколько искал работу

### Шаг 10. Запись в трудовую книжку

- [ ] Бумажная книжка (если есть)
- [ ] Электронная (СЭТД с 2020 — обязательно)
- [ ] Формулировка: "Уволен в связи с сокращением численности или штата работников организации, пункт 2 части первой статьи 81 Трудового кодекса Российской Федерации"

### Шаг 11. Уведомление центра занятости

- [ ] При массовом сокращении (> 50 чел / 1 мес ИЛИ > 200 чел / 2 мес ИЛИ > 500 чел / 3 мес — критерии разные по регионам) — уведомление **за 3 мес** до сокращения

→ В нашем случае (5 человек) — не массовое, но рекомендую уведомить **за 2 мес** для bonus protection.

### Шаг 12. Хранение документов

- Кадровое дело сохраняется **50 лет** (с 2003) / 75 лет (до 2003) — ФЗ-125 «Об архивном деле»
- Все уведомления + предложения + отказы + приказы + расчёты — в архиве

## Risk assessment

| Сценарий | Probability | Exposure |
|----------|-------------|----------|
| Уволенный работник восстановления требует | 30-40% (любой случай) | 1 СЗП × количество мес до решения суда |
| Если процедура соблюдена идеально | 10-15% probability проигрыша | ~3-5 СЗП per worker |
| Если есть процедурные нарушения | 60-80% probability проигрыша | СЗП × срок + 50-100k морал.вред + 50% штраф |
| Кузнецова уволена (хотя нельзя) | **95%+ восстановление** | СЗП за весь период + морал.вред + штрафы КоАП |
| Орлов (пре-пенс) — если плохо обосновано | 50% suit + criminal exposure | КоАП 5.27 + УК 144.1 |

## Recommendation

1. **Исключить Кузнецову и Орлова** из текущего сокращения
2. **Document business case** для остальных 3 (Иванов, Петров, Сидорова)
3. **Альтернатива для Кузнецовой/Орлова:** соглашение сторон ст.78 ТК с компенсацией 5-8 окладов
4. **Сроки запустить:** today minus 2 мес buffer — если хотим закрыть к декабрю, уведомления должны быть подписаны не позднее октября
5. **Engagement outside-адв ФПА:** рекомендую для finalize плана и review всех уведомлений

## Cost estimate (для 3 безопасных увольнений)

- Выходные пособия: 3 × 1 оклад = ~300k ₽
- Доп.пособия (если не трудоустроятся): 3 × 2 оклада = ~600k ₽
- Outside-адв review: ~100k ₽
- **Total budget:** ~1М ₽

vs. Risk если ошибиться: восстановление + СЗП × 5 + штрафы = **~10М ₽ exposure**
```
