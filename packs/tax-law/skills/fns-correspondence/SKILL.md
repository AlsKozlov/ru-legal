# /fns-correspondence

## Назначение

Ответы на различные формы коммуникации с ФНС вне рамок акта проверки.

## Виды требований

### A. Требование о представлении документов (ст.93)

**Когда:** В рамках камеральной / выездной проверки.

**Срок ответа:** **10 рабочих дней** (продление возможно по обоснованному ходатайству).

**Что может запрашиваться:**
- Договоры с контрагентами
- Первичные документы (накладные, акты, счёт-фактуры)
- Реестры
- Финансовая отчётность
- Иные документы, имеющие отношение к проверке

### B. Требование о представлении документов / информации по конкретному контрагенту (ст.93.1) — "встречная проверка"

**Когда:** ФНС проверяет другого налогоплательщика и хочет узнать про наше сотрудничество.

**Срок ответа:** **5 рабочих дней** (короче чем по ст.93)

### C. Уведомление о вызове в налоговый орган (ст.31)

**Когда:** Для дачи объяснений по налоговым вопросам.

**Действие:** Обязательно явиться (или направить представителя с доверенностью).

### D. Запросы от ФНС по выписке банка (ст.86)

**Адресат:** Банк (не налогоплательщик напрямую).
**Срок реагирования банка:** 3 раб.дня.
**Information disclosed:** Движение средств, остатки.

### E. Сообщения АСК НДС-2 о расхождениях

**Когда:** Auto-flagged разрыв в цепочке.
**Срок:** Сначала **уведомление** — добровольное representation; через 10 раб.дней — formal требование.

### F. Уведомление о приостановлении операций по счетам (ст.76)

**Когда:** За неуплату налога / непредставление декларации / задолженность.
**Срок:** Действует с момента вручения.
**Action:** Срочно — либо устранить причину, либо обжалование.

## Workflow

### Шаг 1. Identify тип + проверить срок

```markdown
## Поступившее требование

- **Тип:** [ст.93 / ст.93.1 / ст.31 / иное]
- **Реквизиты:** № [N] от [DD.MM.YYYY]
- **Дата вручения:** [DD.MM.YYYY] (с этой даты считать срок!)
- **Срок ответа:** [DD.MM.YYYY]
- **Контактное лицо в ИФНС:** [ФИО + телефон + email]
```

### Шаг 2. Анализ запроса

```markdown
## Что запрашивается

[Перечень запрашиваемых documents / information дословно]

## Анализ

- В рамках какой проверки (если ст.93): [...]
- Период запроса: [...]
- Specifically — какие документы / какая инфо нужна
- Можем ли мы предоставить? Все / частично / нет
- Содержит ли информация ПДн третьих лиц? (если да — minimize disclosure)
- Содержит ли коммерческую тайну? (можно request режим КТ для confidentiality)
```

### Шаг 3. Decision tree

#### Сценарий A: Полностью предоставляем

```
- Все документы есть
- Никаких privilege issues
- Срок реалистичен
→ Submit в срок
```

#### Сценарий B: Частично

```
- Часть documents отсутствует / уничтожена (срок хранения истёк)
- Часть содержит privileged information (адв.тайна — если outside-адв ФПА involved)
- Часть outside scope запроса
→ Submit что есть + cover letter с объяснением
```

#### Сценарий C: Ходатайствовать о продлении

```
- Объём чрезмерен
- Сложно физически подготовить в срок
- Документы в архиве удалённом
→ Письменное ходатайство о продлении в установленный срок (минимум за 3 раб.дня до истечения)
```

#### Сценарий D: Не предоставлять / ограничить

```
- Запрос outside legal authority (не относится к проверке)
- Документы не существуют
- Документы содержат privileged information
→ Cover letter с обоснованием + альтернативное предложение если возможно

⚠ Игнорировать запрос — НЕЛЬЗЯ. Штрафы по ст.126 (5,000 руб за каждый документ, до 200,000 руб) или ст.129.1 (5,000 руб).
```

### Шаг 5. Submit + tracking

#### Способы

1. **ЛК налогоплательщика на сайте ФНС** — recommended (instant подтверждение)
2. **Электронный документооборот через оператора ЭДО** — для крупных
3. **Лично в канцелярию** — с проставлением штампа
4. **По почте РФ** заказным с описью — последний resort

#### После

- Log дату подачи + копию у себя
- Track ответ ФНС (получили? уточняющие запросы? решение по проверке?)
- Если запрос followup — иметь historic correspondence ready

## Уголовные риски (УК ст.198, 199)

Если в ходе ответа на запросы становится понятно, что налоговые правонарушения
повлекут уголовные дела:

- **ст.198** — для физ.лица (или ИП) — уклонение от уплаты крупного / особо крупного размера
- **ст.199** — для ЮЛ — уклонение через ЮЛ в крупном / особо крупном размере

→ Engage outside-адв ФПА **немедленно**
→ Не давать показаний под доверенностью без адвоката
→ ст.76.1 УК — освобождение от ответственности при возмещении ущерба **до возбуждения уголовного дела** — может быть strategically applied

## Common pitfalls

### 🔴 Critical

- **Игнорировать требование** → штрафы; presumption against vас в проверке
- **Confessional language** в cover letter → закрепляет позицию ФНС
- **Не запросить продление** при недостатке времени → low-quality ответ

### 🟡 Medium

- Submit too much (over-disclosure) — особенно если есть privileged information
- Submit "raw" без cover letter — ФНС может неправильно интерпретировать
- Не сохранить copy ответа — at дальнейших stages не сможете отстаивать что предоставили

## Что НЕ делает

- Не подает за налогоплательщика (нужна подпись)
- Не handles формальную проверку через все этапы — это `/tax-audit-response`
- Не negotiates с ФНС informally — это outside-адв / налоговый advisor

## Why RU-unique

US IRS information document requests (IDRs) — different framework (informal but with subpoena power). РФ — formal step-by-step procedure с specific deadlines.

**Правовая основа:**
- НК ст.23 (обязанности налогоплательщика)
- НК ст.31 (полномочия налоговых органов)
- НК ст.86 (выписки банков)
- НК ст.88-89 (проверки)
- НК ст.93 (требование о документах в проверке)
- НК ст.93.1 (встречная проверка)
- НК ст.76 (приостановление операций)
- НК ст.126, 129.1 (штрафы)
- УК ст.198, 199 (уклонение)
