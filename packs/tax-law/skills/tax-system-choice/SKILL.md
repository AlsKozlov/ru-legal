# /tax-system-choice

## Назначение

Анализ — какая система налогообложения оптимальна. Output: рекомендация +
обоснование + comparison таблица.

## Pre-flight (input parameters)

- **Тип юр.лица:** ООО / АО / ИП / физ.лицо
- **Прогноз выручки:** [сумма годовая]
- **Прогноз расходов:** [сумма / % от выручки]
- **Численность работников:** [N]
- **Контрагенты:** B2B (платят НДС) / B2C (преимущественно)
- **ОКВЭД основной:** [для исключений из УСН / ПСН]
- **Регион регистрации:** [некоторые регионы — пониженные ставки УСН]
- **Сфера деятельности:** торговля / услуги / производство / IT / сельхоз / иное

## Основные системы — спецификация (2026)

### 1. ОСНО (Общая система)

**Кто применяет:** по default; нет ограничений по обороту / численности.

**Налоги для ЮЛ:**
- НДС: 20% (стандарт), 10% (некоторые товары), 0% (экспорт)
- Налог на прибыль: 20% (3% федеральный + 17% региональный)
- Налог на имущество: 2.2% от среднегодовой стоимости основных средств (некоторые с кадастровой)
- Земельный, транспортный — по отдельным расчётам
- НДФЛ как агент по работникам: 13% (15% > 5М)
- Страх.взносы: 30% от ФОТ

**Налоги для ИП на ОСНО:**
- НДС 20%
- НДФЛ 13/15% от прибыли (с учётом профессиональных вычетов)
- Имущественный налог
- Земельный / транспортный
- Страх.взносы за себя (фикс) + за работников

**Pros:** Нет ограничений; можно зачитывать НДС с покупок (важно для B2B)
**Cons:** Самая сложная отчётность; наивысшая налоговая нагрузка

### 2. УСН "доходы" 6%

**Лимиты:**
- Выручка ≤ **251.4 млн руб/год** (2026, корректируется на дефлятор)
- Численность ≤ 130 человек
- Остаточная стоимость ОС ≤ 150 млн

**Налог:** 6% от полученных доходов (cash basis для определения момента)

**Уменьшение:**
- Можно уменьшить налог **до 50%** на страх.взносы (для ЮЛ; для ИП без работников — до 100% если фикс.взносы)

**Замещает:**
- Налог на прибыль ЮЛ
- НДС (нет!)
- Налог на имущество (кроме объектов с кадастровой стоимостью)

**Pros:** Простота учёта; низкая ставка
**Cons:** Без вычета НДС → не подходит для B2B; нет учёта расходов

### 3. УСН "доходы-расходы" 15%

**Лимиты:** те же, что УСН-доходы

**Налог:** 15% от прибыли (доходы минус расходы) — closed list расходов в ст.346.16

**Минимальный налог:** 1% от доходов (если 15% от прибыли меньше)

**Pros:** Учитываются расходы; подходит для торговли с маржой
**Cons:** Сложнее учёт (нужно substantiate расходы); тоже без НДС

### 4. ЕСХН (Сельхозналог)

**Кто:** Производители с/х продукции, у которых доля выручки от с/х > 70%

**Налог:** 6% от прибыли

**Pros:** Низкая ставка; узкая специализация
**Cons:** Только сельхоз

### 5. АУСН (Автоматизированная УСН)

**Кто:** Малый бизнес в пилотных регионах (Москва, Подмосковье, Татарстан, Калужская — на 2026 расширяется)

**Лимиты:**
- Выручка ≤ **60 млн руб/год**
- Численность ≤ **5** работников

**Налог:**
- "Доходы" 8%
- "Доходы-расходы" 20% (но минимум 3% от доходов)

**Уникально:**
- **Страх.взносы за работников включены в ставку** (т.е. не платятся отдельно)
- Декларации **не подаются** (ФНС считает автоматически по выпискам банка)

**Pros:** Минимум отчётности; predictable cost
**Cons:** Узкие лимиты; пилотные регионы

### 6. ПСН (Патентная система) — только для ИП

**Кто:** ИП, виды деятельности по перечню (ст.346.43 НК) — torговля розничная, услуги, и т.п.

**Лимиты:**
- Выручка ≤ **60 млн руб/год**
- Численность ≤ **15** работников

**Налог:** Фиксированная сумма за патент (potential income × 6%, где potential income — устанавливается региональным законом)

**Pros:** Predictable; простота
**Cons:** Только ИП; узкий список ОКВЭД; не все регионы все ОКВЭД покрывают

### 7. НПД (Налог на профессиональный доход — самозанятые)

**Кто:** Физ.лица + ИП без работников, выручка ≤ 2.4 млн/год

**Ставка:**
- **4%** при работе с физ.лицами
- **6%** при работе с ЮЛ / ИП

**Ограничения:**
- Запрет если был работником этого заказчика < 2 лет назад
- Нет работников
- Не реализует подакцизные товары / товары с маркировкой
- Не агентский договор от имени принципала
- Не сдача недвижимости в коммерческих целях (для non-нежилого imovement)

**Pros:** Минимальная ставка; чек через "Мой налог" в смартфоне
**Cons:** Узкие лимиты; нет страх.взносов → нет пенсионных прав

## Workflow

```markdown
**Бизнес:** [ООО / ИП / физ.лицо]

**Численность:** [N]
**ОКВЭД:** [код]
**B2B vs B2C:** [%]
**Регион:** [для пониженных ставок]
```

### Шаг 2. Filter applicable systems

| Система | Применима? | Reason |
|---------|-----------|--------|
| ОСНО | Да всегда | По default |
| УСН доходы 6% | [Y/N] | Если выручка / численность / ОС в лимитах |
| УСН доходы-расходы 15% | [Y/N] | То же |
| ЕСХН | [Y/N] | Если сельхоз > 70% |
| АУСН | [Y/N] | Если регион + лимиты |
| ПСН | [Y/N] | Только ИП + ОКВЭД из перечня |
| НПД | [Y/N] | Только физ.лицо/ИП без работников + < 2.4 млн |

### Шаг 3. Calculate налоговую нагрузку per применимым

Для каждой:
- Тotaл налогов (НДС + налог на прибыль/УСН + имущественный + страх.взносы + НДФЛ работников)
- Сложность учёта (low / medium / high)
- Cash flow impact (предоплата / postpay)

### Шаг 4. Specific scenarios

#### Scenario A: IT-стартап с B2B клиентами

Часто оптимально:
- **ОСНО + IT-аккредитация** (ставка налога на прибыль 0% до 2030, страх.взносы 7.6% при определённых условиях)
- Дешевле УСН в B2B контексте т.к. контрагенты-юр.лица хотят НДС-счёт-фактуры

#### Scenario B: Малая торговля B2C

Часто оптимально:
- **УСН доходы-расходы 15%** (учёт расходов важен)
- ИЛИ **ПСН** для ИП (predictable cost)

#### Scenario C: Самозанятый-фрилансер

- **НПД** (4% / 6%) — простота
- Альтернатива — ИП на УСН доходы 6% (если выручка приближается к 2.4 млн)

#### Scenario D: Производство

- **ОСНО** (для зачёта НДС с сырья) или
- **УСН доходы-расходы 15%** если маржа высокая и закупки без НДС (например, от УСН-щиков)

#### Scenario E: Сельхозпроизводитель

- **ЕСХН** — обычно оптимально
- УСН — альтернатива при transition periods

## Transitions / requirements

### Если переход на УСН 15%

- [ ] Заявление в ФНС о переходе **до 31 декабря** предыдущего года
- [ ] Подача "Уведомления о переходе на УСН" (форма 26.2-1)
- [ ] Учёт переходных операций (НДС: списать остатки в Q4)
- [ ] Перейти на упрощённый учёт расходов (книга учёта доходов и расходов)

### Watch-outs

- Если выручка превысит 251.4 млн (2026) — automatic переход на ОСНО + retroactive recalc
- Если численность > 130 — то же
- Если ОС > 150 млн — то же
- **При превышении 188.55 млн (≈ 75% от лимита)** — повышенная ставка (для УСН доходы — 8%, для УСН доходы-расходы — 20%) на квартал превышения и далее

## Outside consultation recommended

[Когда рекомендуется engage outside-консультанта — например, при transition между системами с complex transitional rules; при IT-аккредитации process]

## Что НЕ делает

- Не подаёт заявление в ФНС (это сам бизнес / аутсорс)
- Не делает retroactive recalc — это бухгалтерия
- Не handles transition между налоговыми режимами в текущем году

## Why RU-unique

US tax system — single federal + state layers, не tier-based small business systems. РФ tax system — multi-tier с specific small business preferences.

**Правовая основа:** НК главы 21 (НДС), 25 (налог на прибыль), 26.1 (ЕСХН), 26.2 (УСН), 26.5 (ПСН), ст.346.43 (виды для ПСН), ФЗ-422 (НПД)
