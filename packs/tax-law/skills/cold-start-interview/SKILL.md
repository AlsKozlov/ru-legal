# /cold-start-interview (tax-law)

## Workflow

### Шаг 1. Тип бизнеса + налоговая система

- ООО / АО / ИП / самозанятый
- Текущая система (см. matrix ниже)
- Год перехода / запланирован ли переход

#### Налоговые системы для ЮЛ

| Система | Когда применима | Ставки | Главное преимущество | Главный минус |
|---------|-----------------|--------|----------------------|----------------|
| ОСНО | По default; нет ограничений | НДС 0/10/20%, налог на прибыль 20%, иные | Без ограничений по обороту | Сложная отчётность, наивысшая налоговая нагрузка для малого бизнеса |
| УСН доходы 6% | Выручка < 251.4 млн (2026), численность < 130 | 6% от доходов (МОЖНО уменьшать на страх.взносы) | Простота | Без вычета НДС → не подходит для B2B с НДС-плательщиками |
| УСН доходы-расходы 15% | То же + большая доля расходов | 15% от прибыли | Подходит для торговли с маржой | Сложнее учёт расходов |
| ЕСХН | Сельхоз > 70% выручки | 6% от прибыли | Для сельхоза — оптимально | Узкая применимость |
| АУСН | < 5 работников, выручка < 60 млн | 8% доходы / 20% доходы-расходы | "Бесплатные" страх.взносы (учтены в ставке) | Узкие лимиты + пока 4 региона |

#### Для ИП дополнительно

| Система | Когда | Ставка |
|---------|-------|--------|
| ПСН (патент) | < 60 млн/год + < 15 работников | Фикс. сумма за патент (зависит от ОКВЭД + региона) |
| НПД (самозанятый) | < 2.4 млн/год + нет работников | 4% (с физ.лиц) / 6% (с ЮЛ) |

## Why RU-unique

Anthropic не имеет налогового pack-а. US tax law (IRC) — federal + state — fundamentally different framework от РФ НК с разделами + регионального компонента.

**Правовая основа:**
- НК РФ (полностью)
- Особенно: ст.138-140 (досудебка), раздел V.1 (transfer pricing), ст.119 (декларации) + ст.122 (неуплата) + ст.123 (агент НДФЛ) штрафы
