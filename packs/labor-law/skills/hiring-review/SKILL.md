# /hiring-review

## Назначение

Pre-signing review трудового договора / offer letter для compliance с ТК. Flags:
- Missing обязательных условий (ст.57) → договор оспорим
- Незаконный срочный (без ст.59 grounds) → переквалификация в бессрочный
- Непропорциональный испытательный срок → ст.71 защита работника
- Должность не в штатном расписании / без должн.инструкции
- Conflict с ЛНПА (правила, положение об оплате)
- Скрытые риски (несамозанятый под видом ГПД, гражданско-правовая mask ТД)

## Pre-flight

Чтобы анализ был ценным, нужно:
- **Тип отношений:** трудовой / ГПД / самозанятый (НПД)? → если не ТД — route к
  `/worker-classification`
- **Категория работника:** инвалид / беременная / несовершеннолетний / иностранец / совместитель / удалёнщик / нет
- **PROFILE.md загружен** (для региональных коэффициентов, штатного, ЛНПА)

## Workflow

### Шаг 1. Identify базовые параметры договора

- **Стороны:** работодатель + работник
- **Должность:** [название] — соответствует штатному расписанию? Должн.инструкция есть?
- **Дата начала:** [DD.MM.YYYY]
- **Срок:** бессрочный / срочный (если срочный — основание ст.59?)
- **Испытательный срок:** есть / нет / какой длительности
- **Зарплата:** оклад + надбавки + премии (структура)
- **Место работы:** офис / удалённо / разъездной / гибрид
- **Режим:** 40 час/нед / неполное / гибкий / сменный

### Шаг 2. Обязательные условия (ст.57 ТК) — checklist

| # | Условие | Есть? | Заметки |
|---|---------|-------|---------|
| 1 | ФИО работника + ПДн (паспорт) | | |
| 2 | Наименование работодателя + ИНН | | |
| 3 | Сведения о представителе работодателя (если подписывает не директор) | | если подписывает HR — доверенность |
| 4 | Место и дата заключения | | |
| 5 | Место работы (адрес или регион + удалёнка) | | для дистанционных — ст.312.2 ч.6 specifics |
| 6 | Трудовая функция (должность + квалификация + конкретный вид работы) | | без этого договор оспорим |
| 7 | Дата начала работы | | если срочный — также конец |
| 8 | Условия оплаты (оклад/ставка + доплаты + надбавки + поощр.) | | конкретные суммы / формулы |
| 9 | Режим рабочего времени и времени отдыха (если отличается от ЛНПА) | | |
| 10 | Гарантии и компенсации за тяжёлые / вредные условия (если применимо) | | |
| 11 | Условия о характере работы (разъездная, в пути и т.п.) | | |

**Если что-то отсутствует** — recommend добавить, договор без обязательных
условий может быть оспорен (хотя сам факт работы → отношения признаются
трудовыми).

### Шаг 3. Срочный договор — особый scrutiny

Если срочный (< 5 лет), **обязательно** указать **основание ст.59 ТК**:

**Часть 1 ст.59 — заключается без согласия работника:**
- Замещение временно отсутствующего работника (декрет, командировка, болезнь)
- Сезонные работы
- Работа за границей в командировке
- Работа за пределами обычной деятельности (расширение производства > 1 года)
- Работа для конкретного проекта (если работа не может быть продолжена бессрочно)
- Стажировка / обучение
- Альтернативная гражданская служба
- Иное

**Часть 2 ст.59 — по соглашению сторон (т.е. с согласия работника):**
- Работодатель — субъект малого предпринимательства (< 35 чел; в розничной торговле / бытовых услугах < 20)
- Пенсионеры по возрасту
- Работа на Крайнем Севере (если связано с переездом)
- Творческие работники (СМИ, кино, театры)
- Руководители, заместители, главбух
- Совместители
- Иное

**Flag** если:
- Основание не указано → переквалификация в бессрочный (ст.58 ч.5)
- Основание формальное ("по соглашению сторон" без часть 2 ст.59 grounds) → переквалификация
- Срочный с работником, который замещает себя (commonly suspicious с ИТ-специалистами на 1 год без grounds) → ст.59 violation

### Шаг 4. Испытательный срок (ст.70-71)

**Ограничения:**
- Max 3 месяца для обычных работников
- Max 6 месяцев для **топ-менеджеров** (руководитель, его заместители, главбух, главы филиалов / представительств / структурных подразделений)
- Max 2 недели для срочных до 6 месяцев

**Запрещён для:**
- Беременных и женщин с детьми до 1.5 лет
- Несовершеннолетних
- Молодых специалистов (1 год со дня окончания ВУЗ/СПО + first job по специальности)
- Избранных на выборную должность
- Приглашённых из другой организации по соглашению
- Заключающих ТД на срок до 2 мес
- Иных категорий по ФЗ / колдоговору

**Flag** если:
- Установлено вопреки запрету → испытание считается не установленным
- Длительность > limit → излишек не действует
- Цель испытания не сформулирована → проблема для увольнения по ст.71
- Зарплата на испытании ниже штатной → дискриминация (ст.3 ТК)

### Шаг 5. Особые категории

#### Иностранный работник

- Должность — соответствует разрешению на работу / patentu?
- Если ВКС (высококвалифицированный) — пометка в договоре
- Уведомление МВД (УВМ) в 3 раб.дня с заключения — обязательное (ст.13 ФЗ-115 «О правовом положении иностранных граждан»)
- Срок ТД не превышает срок миграционного документа
- Иностранец без разрешения / patenta при необходимости — штраф 250-800k за work-а (КоАП ст.18.15)

#### Несовершеннолетний (14-18 лет)

- Письменное согласие родителя (до 16) + органа опеки (до 15)
- Медосмотр обязателен (ст.69, 266)
- Сокращённое раб.время (ст.92): до 16 — 24 ч/нед, 16-18 — 35 ч/нед
- Запрет на: тяжёлый / вредный / ночь / сверхурочные / командировки (ст.265)

#### Удалёнка (дистанционная работа)

- Указать что работа дистанционная (ст.312.1)
- Условия использования оборудования (работник или work-дателя ст.312.6)
- Порядок и сроки выплаты компенсации за свои инструменты
- Способы взаимодействия (электронная подпись, мессенджер)
- Возможные основания доп.увольнения ст.312.8 (взаимодействие < 2 раб.дня = повод)

#### Совместитель (ст.282-288)

- Указать что это совместительство (внешнее / внутреннее)
- Зарплата за фактически отработанное (ст.285)
- Max ½ нормы времени (т.е. 20 ч/нед для full = 40 ч)
- Особое основание для увольнения — приём work-а, для которого работа основная (ст.288)
- Запрет на совместительство в опасных условиях, для несовершеннолетних, и т.п. (ст.282)

### Шаг 6. Зарплата и compensation

- **Оклад / тарифная ставка** — указан конкретно (не "по штатному")
- **Региональные коэффициенты** — добавлены если регион северный / Чернобыль (см. PROFILE.md)
- **Премии** — структура (положение об оплате применимо?)
- **Доплаты / надбавки** — за вредные условия, за стаж, за разъездной характер
- **МРОТ check** — итого не менее регионального МРОТ (с 2026: ~22k ₽ федеральный, региональные могут быть выше)
- **Выплата** — 2 раза в месяц обязательно (ст.136 ТК) — конкретные даты
- **Способ выплаты** — на банковскую карту (с письменного согласия и указанием банка)

### Шаг 7. Risk patterns (high)

#### 🔴 Гражданско-правовая mask трудовых отношений

Признаки:
- "ГПД" с регулярной зарплатой 2 раза/мес
- Подчинение режиму компании
- Использование оборудования компании
- Личное выполнение (не передача под/подрядчику)
- Длительность > 6 мес

→ Высокий риск переквалификации (по решению суда / ГИТ → ст.19.1 ТК; штрафы по
КоАП ст.5.27 + доначисление НДФЛ + страх. взносов + пеня).

#### 🔴 Самозанятый (НПД) под видом

С 2019 распространённый pattern: компания вместо ТД заключает договор оказания
услуг с самозанятым.

**ФНС / суд переквалифицируют если:**
- Бывший сотрудник перешёл в НПД < 2 лет назад (запрет по ФЗ-422 ст.6 п.8)
- Один заказчик постоянно
- Регулярные ежемесячные выплаты
- Работа в офисе на оборудовании заказчика

→ Штрафы + страх.взносы + НДФЛ за весь период.

См. `/worker-classification` для deep-dive.

#### 🔴 Зарплата "по штатному" без конкретной суммы

Не allowed (ст.57 п.5). Должна быть конкретная сумма или формула. Иначе договор
не имеет обязательных условий.

#### 🟡 Premium конструкция

Если 80% дохода — премии "по усмотрению работодателя", оклад — 30k:
- Лишение премии де-факто = снижение зарплаты
- Может быть оспорено как недопустимое изменение существенных условий (ст.74)
- Recommend: фиксированный оклад > МРОТ + chiselled премии по measurable KPI

#### 🟡 Mandatory non-compete без compensation

ТК не предусматривает non-compete (post-termination). Подписи "обязуюсь не работать у конкурента" без компенсации = unenforceable (КС РФ постановление 23-П / 2018 + practice).

**Workable:** обязательство соблюдать коммерческую тайну (98-ФЗ) — это до 5 лет с компенсацией (если установлено положением о КТ + договор о неразглашении).
