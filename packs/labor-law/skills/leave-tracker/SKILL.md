# /leave-tracker

## Назначение

Comprehensive view + analysis отпусков / больничных. Unlike `/log-leave` (quick
single entry), это:
- Compliance review (соблюдение графика отпусков, лимитов, ст.124 переноса)
- Risk flags (накопленные отпуска > 2 года = violation, не предоставлен 14-дневный
  непрерывный отрезок)
- СФР interactions (электронные больничные, decретные пособия, ОТмена/перерасчёт)
- Декретные timeline management
- Audit готовность к ГИТ-проверке

## Workflow

### Шаг 1. Identify scope

- **Per-employee:** анализ конкретного сотрудника
- **Period review:** что было в Q4 / годовой
- **Compliance scan:** все ли в графике, неиспользованные отпуска, риски
- **Pre-audit:** подготовка к плановой ГИТ проверке

### Шаг 2. РФ-specific tracking categories

#### A. Ежегодный основной (ст.115 ТК)

- **Минимум 28 календарных дней**
- Расчёт: 2.33 дня за каждый месяц работы (28/12)
- Право возникает после 6 мес работы (ст.122), кроме исключений
- Использование — по **графику отпусков** (ст.123), утверждается до 17 декабря на следующий год с учётом мнения профсоюза

#### B. Дополнительные отпуска

| Категория | Доп.отпуск | Норма |
|-----------|------------|-------|
| Работники с ненормированным днём | min 3 кал.дня | ст.119, локальный акт |
| Северные / приравненные | от 8 до 24 кал.дней (по местности) | ст.321 |
| Чернобыльская зона | 14 кал.дней | закон 1244-1 |
| Вредные / опасные условия (3-4 класс СОУТ) | min 7 кал.дней | ст.117 |
| Спецхарактер работы (мед, образование) | сектoральные нормы | ст.118 |

#### C. Разделение и переносы (ст.124-125)

- **Одна часть ≥ 14 кал.дней непрерывно** — обязательно (ст.125 ч.1)
- Другие части — по соглашению, любой длины
- **Перенос на следующий год** допустим **только в исключительных случаях**, не позднее 12 мес после окончания года, за который полагался отпуск (ст.124 ч.4)
- **Запрет переноса > 2 лет подряд** — для несовершеннолетних и для work-в во вредных условиях; для остальных — фактически правонарушение

#### D. Больничный (ст.183 ТК + ФЗ-255 «Об обязательном соц.страховании»)

- С 2022 — **только электронный** больничный лист (через СФР)
- Первые **3 дня** — за счёт работодателя
- С **4-го дня** — за счёт СФР
- Размер пособия: 60% / 80% / 100% от среднего заработка в зависимости от стажа
  - До 5 лет — 60%
  - 5-8 лет — 80%
  - 8+ лет — 100%
- **Maximum daily** — определяется в зависимости от max базы страх.взносов (с 2026 ~5200 руб/день примерно)

#### E. Декретный (ст.255 ТК)

- 70 + 70 = 140 кал.дней (стандартный)
- 70 + 86 = 156 кал.дней (осложнённые роды)
- 84 + 110 = 194 кал.дня (многоплодная беременность)
- **100% среднего заработка**, max базы страхвзносов
- Не зависит от стажа

#### F. По уходу до 1.5 / 3 лет (ст.256)

- До 1.5 лет — 40% среднего заработка, max ~49,123 руб/мес (2026)
- 1.5-3 года — компенсация 50 руб/мес (минимальная), но при желании пособие не выплачивается
- Доступен matery, отцу, бабушке, дедушке, другому работающему родственнику (на выбор)
- В период декрета — место работы сохраняется (нельзя уволить кроме ликвидации)

#### G. Отпуска без сохранения зарплаты (БСС) — ст.128

- Стандартный по соглашению — любой срок
- **Обязательные** категории (ст.128 ч.2):
  - Участники ВОВ — до 35 кал.дней / год
  - Работающие пенсионеры по возрасту — до 14 кал.дней / год
  - Родители / супруги погибших военных — до 14 кал.дней / год
  - Работающие инвалиды — до 60 кал.дней / год
  - В связи с рождением / регистрацией брака / смертью близкого — до 5 кал.дней
  - Иные по ФЗ / колдоговору

#### H. Учебные (ст.173-176)

- При получении первого образования (только если аккредитованная программа)
- Различные нормы: бакалавриат 50 кал.дней/год, ВУЗ магистратура, СПО, аспирантура
- Оплачиваются работодателем (если first education) или БСС (если second)

### Шаг 3. Compliance checklist (annual)

```markdown
## Annual Leave Compliance — [год]

### График отпусков
- [ ] Утверждён до 17 декабря предыдущего года
- [ ] (Если профсоюз) учтено мнение профсоюза
- [ ] Работники ознакомлены под подпись
- [ ] Изменения в течение года оформлены приказами

### Использование отпусков

| Работник | Положено всего | Использовано | Перенос | Risk flags |
|----------|----------------|--------------|---------|------------|
| Иванов И.И. | 28 + 7 (вредность) = 35 | 14 | 21 (перенос) | 🟡 не использован 14-дн отрезок |
| Петров П.П. | 28 | 0 | 28 | 🔴 не использовано более года — нарушение |
| Сидорова | 28 + 14 (декрет статус) | в декрете | n/a | 🟢 |
| ... | | | | |

### Декретные / по уходу

| Работник | Тип | Период | Статус | Пособие |
|----------|-----|--------|--------|---------|
| Сидорова | Декрет 140 дн | 01.04 — 18.08.2026 | active | 100% СЗ, выплачивается СФР |
| ... | | | | |

### Больничные

- Общая статистика: [N] листов, [M] дней — за период
- Anomalies: [сотрудники с подозрительно частыми больничными — flag для review]
- Электронные листы all properly registered: Y/N

### СОУТ status

- Last СОУТ done: [DD.MM.YYYY]
- Next due: [DD.MM.YYYY] (каждые 5 лет минимум — ст.8 ФЗ-426)
- Вредные / опасные классы → flag для доп.отпуска / доп.оплаты
```

### Шаг 4. Risk flags (приоритеты)

#### 🔴 Critical

- **Не использовано > 2 года** — нарушение ст.124, ГИТ flag
- **Не предоставлен 14-дневный непрерывный отрезок** при наличии возможности — ст.125 ч.1
- **График отпусков не утверждён** — ст.123, штраф КоАП 5.27
- **Беременная женщина не получила положенный декретный отпуск перед родами** — нарушение

#### 🟡 Желательно исправить

- Накопленные дни > 56 (двойная норма)
- Перенос без оформленных оснований / приказа
- Преждевременный отзыв из отпуска без согласия (ст.125 ч.2)
- Замена части отпуска компенсацией свыше 28-дневного остатка (ст.126 — запрет для беременных, несовершеннолетних, занятых во вредных условиях)

#### 🟢 OK

- Все work-ки идут по графику
- Декретные пособия зарегистрированы корректно
- СОУТ актуальна

### Шаг 5. Action items per identified issue

Для каждого flag — recommended action:

| Flag | Action |
|------|--------|
| Не использовано > 12 мес | Уведомить work-а, предложить план использования; письменно зафиксировать |
| Не использовано > 24 мес | **Срочно** — план до конца года; risk предписания ГИТ; **deнежная компенсация запрещена** кроме увольнения |
| 14-дн отрезок не предоставлен | Запланировать в следующем графике обязательно; уведомить work-а |
| График не утверждён | Срочно утвердить, ознакомить работников |
| Замена > 28 дней компенсацией | Cancel и предоставить отпуск; иначе risk суда |

## Что НЕ делает

- Не рассчитывает средний заработок (это бухгалтерия / 1С)
- Не выпускает приказы (HR system)
- Не submits в СФР электронные листы (HR system)
