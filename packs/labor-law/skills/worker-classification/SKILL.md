# /worker-classification

## Назначение

Determine правильную правовую природу отношений + flag riskи переквалификации.
В РФ это **tripartite distinction**:

| Тип | Регулируется | Налоги | Страх.взносы | Защита ТК |
|-----|--------------|--------|--------------|-----------|
| **ТД** (трудовой договор) | ТК | НДФЛ 13/15% (источник — работодатель) | 30%+ от ФОТ (работодатель) | Полная (отпуска, больничные, ГИТ, и т.д.) |
| **ГПД** (договор подряда / возмездного оказания услуг) | ГК (гл. 37, 39) | НДФЛ 13/15% (источник — заказчик) | Страх.взносы по части (страх. на проф.заболевания иногда нет; по нек. видам — 27.1%) | Нет |
| **НПД** (самозанятый) | 422-ФЗ | 4% (с physical лиц) / 6% (с юр.лиц / ИП) | Нет (за исключением добровольной) | Нет |

Misclassification → ФНС, ГИТ, суд могут переквалифицировать с retroactive consequences.

## Critical РФ норма: ст.19.1 ТК (2014)

> "Признание отношений, возникших на основании гражданско-правового договора,
> трудовыми отношениями может осуществляться: ... 3) **судом** в случае, если
> физическое лицо, являющееся исполнителем по указанному договору, обратилось
> непосредственно в суд..."

Также:
- **ГИТ может выдать предписание** о признании отношений трудовыми (ст.19.1 ч.2)
- **ФНС** проверяет через выездные / камеральные проверки, доначисляет НДФЛ + страх.взносы

**В случае сомнений отношения признаются трудовыми** (ст.19.1 ч.3) — это
load-bearing presumption.

## Pre-flight

- Какой тип договора заявлен? (ТД / ГПД / НПД)
- Какие признаки фактически имеют место? (см. checklist ниже)
- Сколько длится сотрудничество?
- История: был ли исполнитель раньше work-ом этой компании? (важно для НПД)

## Workflow

### Шаг 1. Apply distinguishing criteria

#### Признаки ТРУДОВЫХ отношений (Постановление Пленума ВС РФ № 15 / 2018)

1. **Личное выполнение** работы (нельзя передать под/исполнителю)
2. **Подчинение режиму** организации (приходить в офис, на смены, по графику)
3. **Регулярная зарплата** (2 раза в месяц, аналогично штатным)
4. **Использование оборудования / помещения** работодателя
5. **Включение в производственную деятельность** на постоянной основе
6. **Контроль** работодателя над процессом (а не только результатом)
7. **Длительный характер** (не разовая работа)
8. **Должность** в штатном расписании / описание трудовой функции
9. **Социальные гарантии** фактически предоставляются (отпуск, больничный)
10. **Включение в трудовой коллектив** (корпоративные events, mailrate, badge)

#### Признаки ГПД (правильно оформленного)

1. Конкретный **результат** работы (не процесс)
2. **Сроки выполнения** конкретные
3. **Свобода в способе** выполнения
4. **Возможность привлечения** под/исполнителей (если не запрещено)
5. **Свой инструмент** / собственное оборудование
6. **Самостоятельность** в распределении времени
7. **Оплата по факту** результата / акту
8. **Разовый** или проектный характер

#### Признаки НПД (самозанятого, не маскирующего ТД)

1. Все ГПД признаки (см. выше)
2. **Доход < 2.4 млн руб/год** (ограничение 422-ФЗ)
3. **Несколько заказчиков** (не моно-заказчик регулярно)
4. **Не был** работником этого заказчика в течение **2 лет** до договора (422-ФЗ ст.6 ч.2 п.8)
5. **Не оказывает услуг** в сферах исключений (наёмный труд других — ст.4 ч.2 п.6; адвокатская, нотариальная, и т.п.)
6. **Чеки на каждую оплату** через "Мой налог"
7. **Без подчинения** заказчику organizationally

### Шаг 2. Сравни заявленный тип vs фактические признаки

```markdown
## Classification Analysis

**Заявлено:** [ГПД / НПД / ТД]

### Признаки фактически проявленные

| Признак | Есть? | Источник |
|---------|-------|----------|
| Личное выполнение | ☐ | договор п. / practice |
| Подчинение режиму | ☐ | график / observation |
| Регулярная зарплата | ☐ | расчётные документы |
| Использование оборудования компании | ☐ | observation / договор |
| Длительный (> 6 мес) | ☐ | сроки |
| Должность в штатном | ☐ | штатное |
| Свобода в способе | ☐ | договор |
| Конкретный результат | ☐ | договор / акты |
| ... | | |

### Risk score

- **Высокий риск переквалификации** (5+ признаков ТД): ☐
- **Средний риск** (3-4 признака ТД): ☐
- **Низкий риск** (0-2 признака ТД): ☐
```

### Шаг 3. Specific high-risk patterns

#### 🔴 Pattern A: "Бывший работник на НПД"

- Был work-ом компании ТД
- Уволился (любая причина)
- Через < 2 лет заключил договор на НПД
- Выполняет похожую работу

**Risk:** ФНС моментально переквалифицирует. Запрет 422-ФЗ ст.6 ч.2 п.8 — explicit. Штрафы + взносы + НДФЛ retroактивно.

**Action:** STOP. Engage outside-адв перед заключением. Либо ждать 2 года, либо
оформить ТД (если это фактически трудовые отношения).

#### 🔴 Pattern B: "Армия самозанятых для core business"

- 20+ самозанятых выполняют core business функции (например, курьеры доставки)
- Все работают одинаково, контролируются по KPI
- Ежемесячные регулярные выплаты

**Risk:** Высокий риск массовой переквалификации (case против Яндекса / Wildberries
есть в practice). Штрафы за каждого + страх.взносы.

**Action:** Review business model. Либо ТД, либо реальная аутсорс-фирма (не
индивидуальные НПД).

#### 🔴 Pattern C: "ГПД с регулярной зарплатой 2 раза/мес"

- ГПД на год / без срока
- Оплата 1 и 15 числа каждого месяца, фиксированная сумма
- Подчинение режиму офиса
- Использование оборудования компании

**Risk:** Бесспорная переквалификация в суде по жалобе или ГИТ.

**Action:** Заключить ТД с retroactive datе — либо ожидать претензии.

#### 🟡 Pattern D: "Удалённый IT-специалист на ГПД"

- Программист работает удалённо, на своём оборудовании
- ГПД на конкретный продукт (модуль)
- Оплата по этапам

**Risk:** Medium. Если конкретный результат + свобода в способе — может быть OK.
Иначе risk.

**Action:** Review договор. Strengthen "конкретный результат" + "свобода способа".
Если фактически работает full-time под management — оформить ТД.

### Шаг 4. Налоговые / страх.взносов последствия misclassification

При переквалификации **ГПД / НПД → ТД** ретроактивно (за весь период до 3 лет):

| Налог / взнос | За кого платит | Размер |
|---------------|----------------|--------|
| НДФЛ | Работодатель (как налоговый агент) | 13% (или 15% при доходе > 5 млн) |
| Страх.взносы (ПФР + ОМС + ОСС) | Работодатель | ~30% от ФОТ |
| Страх.взнос на травматизм (СОУТ-зависимый) | Работодатель | 0.2-8.5% |
| Штраф за невыполнение функций налогового агента (НК ст.123) | | 20% от суммы НДФЛ |
| Штраф за неуплату страх.взносов (НК ст.122) | | 20-40% от взносов |
| **Пени** | | СтавкаЦБ × 1/300 за каждый день |

**Plus** административные штрафы по КоАП ст.5.27:
- ч.4 — уклонение от оформления ТД / заключение ГПД, фактически регулирующего трудовые отношения — **штраф 50-100k на компанию**, 10-20k на руководителя
- При повторном — дисквалификация руководителя 1-3 года

```markdown
### Применимая практика

- [Постановление Пленума ВС РФ № 15 / 2018]
- [Конкретный case: например, "Яндекс-курьеры" — суд признал ТД]

## Рекомендация

[Конкретная action — оформить ТД / поправить ГПД формулировки / прекратить НПД и переоформить]

### Если recommend ТД (после ГПД)

- [ ] Заключить ТД с даты [DD.MM.YYYY] (фактической даты начала работы — best practice)
- [ ] Уведомить СФР, ФНС о изменении
- [ ] Пересчитать НДФЛ / страх.взносы (можно сделать корректирующие декларации)
- [ ] Оформить отпуска / больничные / сверхурочные за прошлый период
- [ ] Outside-адв ФПА должен review подход — material exposure

### Если recommend ГПД (с фиксами)

- [ ] Убрать формулировки "должность", "режим работы", "зарплата"
- [ ] Чётко сформулировать "конкретный результат"
- [ ] Изменить периодичность выплат (по актам, не 1 и 15)
- [ ] Стороны не должны быть в trapper'ской зависимости

### Если recommend НПД

- [ ] Проверить 2-year rule (бывший работник)
- [ ] Проверить лимит 2.4 млн руб
- [ ] Несколько заказчиков (не моно)
- [ ] Чеки через "Мой налог"

## Risks if no action

- **Переквалификация** через ФНС / ГИТ / суд
- **Доначисление** НДФЛ + страх.взносы + штрафы + пени за весь период (до 3 лет)
- **Штрафы КоАП** до 100k + дисквалификация при повторе
- **Жалобы work-в** в ГИТ / прокуратуру → проверки, plus reputation risk

## Escalation

При high-risk patterns (A, B, C) — **обязательно** engage outside-адв ФПА до
implementation. Material exposure при крупных портфелях самозанятых.
```
