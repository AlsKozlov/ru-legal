# /trudovaya-inspeksiya-response

## Назначение

Handle проверку или запрос документов от Государственной инспекции труда (ГИТ /
Роструд) или прокуратуры. ГИТ — основной регулятор по ТК; имеет полномочия:
- Запрашивать документы
- Проверять (планово / внепланово, документарно / выездно)
- Выдавать **предписания** (обязательны к исполнению)
- Привлекать к административной ответственности (КоАП ст.5.27 / 5.27.1)
- Передавать материалы прокуратуре для уголовного дела (УК ст.143 — нарушение охраны труда)

## Pre-flight checklist (critical!)

### Шаг 0. Verify legitimacy запроса

- **Письменное решение о проведении проверки** (для проверок) — обязательно по 248-ФЗ
- **ФИО / должность инспектора + удостоверение** — verify через реестр Роструда
- **Основания** — ст.354-360 ТК + 248-ФЗ "О гос.контроле"
- **Если просто запрос документов без проверки** — отдельно регулируется

⚠ **Если что-то не сходится** — фиксировать (видеозапись согласия, акт об отказе допуска), обращаться к outside-адв немедленно.

### Шаг 1. Classify type

| Тип | Длительность | Основание | Что включает |
|-----|--------------|-----------|--------------|
| **Плановая** | До 20 раб.дней (20 дн + 20 дн продление) | План Роструда на год (публикуется до 31 декабря) | Полный аудит compliance |
| **Внеплановая выездная** | 10 раб.дней | Жалоба, угроза жизни, истечение срока предписания | Конкретное основание |
| **Внеплановая документарная** | 10 раб.дней | Тот же, но без выезда | Только document review |
| **Профилактическая** | Без штрафов | 248-ФЗ — новая категория | Консультационная |
| **Контрольная закупка / выездное обследование / etc.** | varies | 248-ФЗ — новые формы | Specifics |

### Шаг 2. Мораторий 2022-2026 status check

С 2022 действует мораторий на плановые проверки субъектов МСП (Постановление
Правительства РФ № 336/2022 + продления). Проверь:

- **Распространяется ли** мораторий на компанию?
- **Если попадает** — плановая проверка незаконна; можно обжаловать.
- **Внеплановые** проверки по жалобам **не** подпадают под мораторий — проводятся.

С 2026 мораторий поэтапно отменяется — verify актуальный статус.

### Шаг 3. Сроки реагирования

| Действие | Срок | Норма |
|----------|------|-------|
| Ответ на запрос документов (без проверки) | **10 раб.дней** | 248-ФЗ ст.79 |
| Документы для проверки (документарная) | **10 раб.дней** | 248-ФЗ ст.72 |
| Обжалование акта проверки | **15 раб.дней** | 248-ФЗ ст.39 |
| Обжалование предписания | **30 кал.дней** | КоАП ст.30.3 (если штраф) / 248-ФЗ |
| Исполнение предписания | По срокам в предписании | 248-ФЗ |

⚠ **Срок начинает идти с дня получения** — фиксировать дату с проставлением штампа.

## Workflow

### Шаг 1. Initial triage

Поступил документ от ГИТ. Что это?

- **Запрос документов** (без указания на проверку) — Step 2A
- **Решение о проверке** — Step 2B
- **Акт проверки** (по результатам) — Step 2C
- **Предписание** об устранении нарушения — Step 2D
- **Постановление о привлечении к админ.ответственности** — Step 2E

### Шаг 2A. Ответ на запрос документов

Запрос обычно конкретный: предоставить ТД работника X, приказы за период Y,
ЛНПА Z.

**Workflow:**

1. **Verify legitimacy** — ст.357 ТК даёт право требовать; должно быть в формальном письме
2. **Collect** запрошенные документы
3. **Review** на наличие ПДн третьих лиц (если есть — сократить или редактировать)
4. **Draft cover letter:**

```markdown
В [Государственная инспекция труда в г./субъекте]
от [Работодатель], ИНН [N], адрес [...]

# Ответ на запрос № [N] от [DD.MM.YYYY]

В ответ на ваш запрос от [DD.MM.YYYY] № [N] предоставляем запрашиваемые
документы:

1. [Документ 1] — на [N] листах
2. [Документ 2] — на [N] листах
3. ...

Документы предоставлены за период [...].

Дополнительно сообщаем:

- [Если есть документы, которые не предоставляются — с обоснованием
  (банковская тайна; нет в наличии; не относится к запрошенному периоду)]

Приложение: документы по тексту на [сумма] листах.

[Подпись руководителя / уполномоченного лица]
[Дата]
```

5. **Submit** через канцелярию с проставлением даты + расписки; или Почтой
   России с описью + уведомлением о вручении

**Дедлайн:** 10 рабочих дней (248-ФЗ ст.79). При сложностях с подготовкой —
**заявить ходатайство о продлении** (минимум за 3 раб.дня до истечения).

### Шаг 2B. Реагирование на решение о проверке

1. **Read решение** (предмет проверки, сроки, состав проверяющих)
2. **Verify** статус инспектора через реестр Роструда
3. **Внутренняя подготовка:**
   - Все запрашиваемые документы готовы и доступны
   - Ответственные лица проинструктированы
   - Outside-адв ФПА уведомлён (если threshold для escalation достигнут)

4. **При выездной проверке:**
   - Право компании: **присутствие представителя при всех проверочных действиях**
   - Право: **ознакомление с материалами проверки**
   - Право: **возражения** на акт проверки
   - Право: **обжалование** в вышестоящий орган / суд

5. **Pre-проверочный аудит (если есть время):**
   - ТД всех работников (соответствие ст.57)
   - ПВТР + положение об оплате
   - График отпусков + отчётность по использованию
   - Штатное расписание + должностные инструкции
   - Журналы инструктажей по охране труда (СОУТ карты)
   - Кадровые приказы за период проверки

### Шаг 2C. Акт проверки (по результатам)

**Не подписывать без review!**

1. **Read carefully** — какие нарушения зафиксированы
2. **Cross-check** факты с документами компании
3. **Возражения** в **15 раб.дней** на акт проверки (248-ФЗ ст.39)

#### Template возражений

```markdown
В [ГИТ субъект, адрес]
от [Работодатель]

# Возражения на акт проверки № [N] от [DD.MM.YYYY]

В соответствии со ст.39 ФЗ-248 "О гос.контроле" представляем следующие возражения
на акт проверки.

## 1. По пункту [N] акта — нарушение [статьи ТК]

**В акте указано:** [цитата нарушения]

**Возражение:** факт не соответствует действительности по следующим основаниям:

1. [Аргумент 1] — со ссылкой на документ-приложение
2. [Аргумент 2]
3. ...

**Документы в подтверждение:**

- [Документ A]
- [Документ B]

Просим **исключить** указанный пункт из акта.

## 2. По пункту [M] акта — ...

[То же]

## Заключение

В свете изложенного просим **пересмотреть** акт проверки и **исключить** пункты
[N], [M] как не соответствующие действительности.

Приложения: документы [N] листов.

[Подпись]
[Дата]
```

### Шаг 2D. Предписание об устранении нарушений

**Предписание обязательно к исполнению!** Неисполнение — отдельный штраф (КоАП
ст.19.5 — 10-20k на компанию + дисквалификация).

**Workflow:**

1. **Read предписание** — какие нарушения, какие сроки исполнения
2. **Assess feasibility** — реально ли исполнить в указанные сроки
3. **Choose path:**

   - **A. Исполнить полностью** — самый низкорисковый
   - **B. Обжаловать (полностью / в части)** — если основания есть (см. ниже)
   - **C. Ходатайствовать о продлении** — если срок недостаточен

4. **Implement** — оформить документы, провести action, отчитаться по исполнению
5. **Submit отчёт** об исполнении с подтверждающими документами в указанный срок

#### Обжалование предписания (если основания есть)

- **Срок:** 30 кал.дней с момента получения (КоАП ст.30.3 если штраф)
- **Куда:** вышестоящий орган Роструда / суд (одновременно или после
  административного обжалования)
- **Forms:** через Госуслуги / напрямую через приёмную

```markdown
[Шапка]

# Жалоба на предписание № [N] от [DD.MM.YYYY]

[Работодатель] не согласен с указанным предписанием по следующим основаниям:

## 1. Нарушение не имело места

[Развернутое объяснение со ссылками на документы]

## 2. Срок исполнения недостаточен

[Если применимо — конкретное обоснование]

## 3. Процессуальные нарушения при проверке

[Если были — конкретные нормы, нарушенные ГИТ]

## Просим

1. Признать недействительным предписание № [N] от [DD.MM.YYYY] в полном объёме
   / в части пунктов [...]
2. Приостановить исполнение предписания на период рассмотрения жалобы

[Подпись, дата, приложения]
```

### Шаг 2E. Постановление о привлечении к админ.ответственности

#### Штрафы КоАП ст.5.27 (общие нарушения ТК)

| Часть | Состав | Штраф (компания) | Штраф (директор) |
|-------|--------|------------------|------------------|
| ч.1 | Нарушение трудового законодательства | 30-50k | 1-5k |
| ч.2 | Повторное по ч.1 | 50-70k | 10-20k или дискв. 1-3 г |
| ч.3 | Уклонение от заключения ТД / заключение ГПД фактически регулирующего ТД | 50-100k | 10-20k |
| ч.4 | Повторное по ч.3 | 100-200k | дискв. 1-3 г |
| ч.5 | Нарушение порядка выплаты зарплаты | 30-50k | 10-20k |
| ч.6 | Зарплата ниже МРОТ | 30-50k | 1-5k (на каждого work-а отдельно!) |

#### Штрафы КоАП ст.5.27.1 (нарушения по охране труда)

| Часть | Состав | Штраф |
|-------|--------|-------|
| ч.1 | Нарушение государственных нормативов охраны труда | 50-80k компания / 2-5k директор |
| ч.2 | Допуск без обучения по охране труда | 60-80k компания / 15-25k директор |
| ч.3 | Допуск без медосмотров | 110-130k компания / 15-25k директор |
| ч.4 | Не обеспечение СИЗ | 130-150k компания / 20-30k директор |

#### Обжалование постановления о штрафе

- **Срок:** 10 суток (КоАП ст.30.3)
- **Куда:** вышестоящему лицу ГИТ или в суд (по выбору)
- **Что:** жалоба на постановление

⚠ **Не пропустить срок 10 дней** — иначе постановление вступает в силу и штраф
становится обязательным к уплате.

### Step 3. Decision tree для типичных ситуаций

```
Поступил документ от ГИТ
├── Запрос документов → Step 2A (10 раб.дней ответ)
├── Решение о проверке
│   ├── Плановая
│   │   ├── Под мораторий? → Обжаловать as незаконную
│   │   └── Не под мораторий → Step 2B (приготовиться)
│   └── Внеплановая
│       ├── По жалобе работника → outside-адв + Step 2B
│       └── По истечении предписания → было ли исполнено? → Step 2D/2B
├── Акт проверки
│   └── Возражения в 15 раб.дней → Step 2C
├── Предписание
│   ├── Исполнимое → Step 2D path A
│   ├── Спорное → Step 2D path B (обжалование)
│   └── Срок недостаточен → Step 2D path C (продление)
└── Постановление о штрафе
    ├── Согласны → оплатить в 60 дн (КоАП ст.32.2)
    └── Не согласны → обжаловать в 10 дн (КоАП ст.30.3)
```

## Escalation thresholds

| Trigger | Action |
|---------|--------|
| Плановая выездная проверка | Engage outside-адв ФПА с самого начала |
| Уголовный нюанс (УК ст.143 — охрана труда; ст.145.1 — невыплата зарплаты) | Outside-адв немедленно + GC notify |
| Штраф > 100k или повторный | Outside-адв + escalate к CFO / CEO |
| Дисквалификация директора в постановлении | Срочно outside-адв + Board notify |
| Прокуратура (а не ГИТ) | Outside-адв немедленно — другой режим |

## Уголовные риски (УК)

### ст.143 — Нарушение требований охраны труда

- Тяжкий вред здоровью / смерть human → УК
- Должностное лицо ответственное за охрану труда
- Наказание: штраф / принудительные работы / лишение свободы до 5 лет

### ст.145.1 — Невыплата зарплаты

- Свыше 2 мес полная невыплата либо свыше 3 мес частичная (более 50%)
- Должно быть умышленным (доказывается ФНС / следствием — есть деньги, но не платит)
- Наказание: штраф до 500k / принуд.работы / лишение свободы до 3 лет

### Действия при уголовном riske

- **Outside-адв ФПА немедленно** (адвокатская тайна для защиты)
- Документировать все объяснения
- Не давать показаний без адвоката
- Cooperate с следствием по advice адвоката

## Что НЕ делает

- Не handles прокуратуру (separate regime — engage outside-адв сразу)
- Не replaces outside-адв в criminal context (УК ст.143 / 145.1)
- Не drafts уголовно-процессуальные жалобы

## Documentation requirements (для audit trail)

После любого interaction с ГИТ — log:
- Дата получения / отправки
- Что получено / отправлено (тип, № документа)
- Кто handled (ФИО, должность)
- Решение / действия принятые
- Outside-адв involvement (если был)
- Outcome
