# /log-leave

## Назначение

Быстрая запись отпуска / больничного / декретного / БСС / учебного в leave tracker.
Не делает полный анализ — это `/leave-tracker`. Просто запись для последующего
review.

## Workflow

### Шаг 1. Спроси минимум информации

- **ФИО / табельный**
- **Тип:** ежегодный / больничный / декрет / отпуск по уходу до 1.5/3 / БСС /
  учебный / донорский / иной
- **Период:** с DD.MM.YYYY по DD.MM.YYYY
- **Основание:** приказ N / больничный лист N / заявление / медзаключение

### Шаг 2. Auto-determine ФСС/СФР статус

| Тип | ФСС/СФР пособие | Заметки |
|-----|-----------------|---------|
| Ежегодный (ст.115 ТК) | Нет — оплата работодателем | 28 кал.дней минимум, по графику |
| Больничный (ст.183 ТК) | Да — после 3 дней первые 3 за счёт работодателя | Электронный лист с 2022 |
| Декрет (ст.255 ТК) | Да — 100% среднего заработка, 140 дней | Не зависит от стажа |
| По уходу до 1.5 (ст.256 ТК) | Да — 40% среднего, max 49,123₽ (2026) | Можно matey / отцу / бабушке |
| По уходу до 3 (ст.256 ТК) | Только компенсация 50₽/мес | Не идёт в стаж work |
| БСС (без сохранения, ст.128 ТК) | Нет | До 14 дней по обязанности (определённые категории) |
| Учебный (ст.173-176 ТК) | Нет — оплата работодателем при условиях | Только если аккредитованная программа |
| Донорский (ст.186 ТК) | Нет — оплата работодателем | 2 дня (день сдачи + дополнительный) |

### Шаг 3. Add запись в tracker

```markdown
## Запись от [DD.MM.YYYY HH:MM]

- **Сотрудник:** Иванов И.И. (таб. №123)
- **Тип:** ежегодный отпуск
- **Период:** 15.06.2026 — 28.06.2026 (14 кал.дней)
- **Основание:** приказ № 45 от 01.06.2026
- **Заявление:** есть, от 25.05.2026
- **ФСС/СФР пособие:** не применимо
- **Заметки:** [любые — например "по графику отпусков", "вызов из отпуска возможен", "присоединение к майским"]
- **Logged by:** [user]
```

### Шаг 4. Flag potential issues

- **Ежегодный < 14 дней непрерывных** — ст.125 ТК требует одну часть ≥ 14 дней
- **Уведомление < 2 недель до начала** — ст.123 ТК (работника)
- **Больничный электронный лист не подтверждён в СФР** — flag для проверки
- **Декретный начался > 30 нед беременности** — late filing, может потерять часть

## Output format

```
✓ Запись добавлена в leave-tracker.md
[Иванов И.И. — ежегодный отпуск 15.06 — 28.06.2026]

⚠ Flag: одна часть ежегодного < 14 дней в текущем графике (см. ст.125 ТК).
Рекомендую review.

Run `/leave-tracker` для полного review.
```

## Что НЕ делает

- Не drafts приказ — это HR system
- Не calculates средний заработок — это бухгалтерия
- Не submits в СФР — это HR system
- Не tracks budget отпускных — это finance
