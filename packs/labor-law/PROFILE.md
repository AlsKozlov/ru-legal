## Standard processes

### Приём на работу

- Документы (ТК ст.65): паспорт + СНИЛС + ИНН + образование + трудовая (или СТД-Р) +
  военный билет
- **Срок оформления:** 3 рабочих дня (ТК ст.67)
- **Испытательный срок:** [N мес, max 3 / 6 ст.70]
- **Подписант ТД:** [генеральный директор / по доверенности / филиал]

### Увольнение

- **Default посыл:** соблюдение **процедуры** > основания
- **Most common основания** (заполнить по практике):
  - По соглашению сторон (ст.78) — _% случаев
  - По собственному желанию (ст.80) — _%
  - По инициативе работодателя (ст.81) — _%
  - Истечение срочного ТД (ст.79) — _%

### Дисциплинарная ответственность

- **Внутренняя процедура** (ст.193): объяснительная (2 дня) → решение → приказ →
  ознакомление под подпись
- **Виды:** замечание / выговор / увольнение (по основаниям ст.81)
- **Срок:** 1 месяц с обнаружения, не позднее 6 мес со совершения

## Sensitive areas

- **Беременные** (ст.261) — увольнение **запрещено** кроме ликвидации (ст.261 ч.1)
- **Несовершеннолетние** — только с согласия ГИТ + комиссии по делам несовершеннолетних
- **Профсоюз представители** (ст.374) — согласие профсоюза при сокращении
