# Подготовка иска в суд общей юрисдикции

## Назначение

Юрист готовит исковое заявление для подачи в суд общей юрисдикции (СОЮ — мировой
или районный суд) по ГПК РФ. Skill возвращает structured draft со всеми
обязательными элементами по ст. 131 ГПК РФ:

1. Правильная шапка (наименование суда, стороны, представитель)
2. Обоснование (фактическое + правовое — со ссылками на актуальные НПА)
3. Исковые требования (чётко сформулированные)
4. Расчёт цены иска + госпошлины
5. Список приложений (по ст. 132 ГПК РФ)
6. Реквизиты для оплаты госпошлины

**Не для:**
- Арбитражные споры (юр.лица vs юр.лица) → используй `claim-draft-arbitration`
- Административные дела → используй `claim-draft-kas` (planned)
- Уголовные дела → отдельный pack (planned)

## Входные данные

**Обязательно:**
- **Краткая фабула** (5-10 предложений) — что произошло, кто что должен
- **Истец** — ФИО / название, контактные данные
- **Ответчик** — ФИО / название, известные контактные данные
- **Категория спора** — выбрать из: трудовой / семейный / наследственный /
  жилищный / возмещение вреда / защита прав потребителей / договорный спор / иное

**Опционально (улучшит точность):**
- **Сумма требований** (для имущественных исков — определит подсудность)
- **Доказательства** (список доступных документов)
- **Досудебный порядок соблюдён** (Y/N — для категорий где обязательно)
- **ИНН ответчика** (если юр.лицо — проверим через ЕГРЮЛ)

## Workflow

### Шаг 1 — Определение подсудности

Подсудность зависит от:
- **Категория спора** (некоторые — исключительная подсудность)
- **Сумма иска** (мировой судья ≤ 50 000 ₽ для имущ., ≥ — районный)
- **Место жительства/нахождения ответчика** (общее правило ст. 28 ГПК)

  Через `pravo.search_npa(query="ГПК РФ ст. 23")` получи актуальную редакцию
  ст. 23 ГПК (мировые судьи). Через `ст. 24 ГПК` — районные суды.

Применяй логику:

- **Трудовой спор:** районный суд по месту жительства истца ИЛИ ответчика
  (на выбор истца, ст. 29 ГПК)
- **Семейные:** районный суд по месту жительства ответчика (исключения:
  с детьми — по месту жительства истца, ст. 29 ГПК)
- **Наследственный:** районный суд по месту открытия наследства
- **Жилищный (споры о праве):** районный по месту нахождения недвижимости
  (ст. 30 ГПК — исключительная)
- **Возмещение вреда:** районный, на выбор истца (по месту жительства истца
  ИЛИ ответчика ИЛИ места причинения вреда, ст. 29 ГПК)
- **Защита прав потребителей:** районный, на выбор истца (по месту жительства
  истца ИЛИ места заключения договора ИЛИ нахождения ответчика, ст. 17 ЗоЗПП)
- **Имущественный ≤ 50 000 ₽:** мировой судья
- **Имущественный > 50 000 ₽:** районный суд

Если подсудность исключительная — фиксируем и идём дальше. Если альтернативная —
рекомендуем наиболее удобный для истца вариант.

### Шаг 2 — Проверка ответчика (если юр.лицо)

  Если ответчик — юр.лицо и есть ИНН:
  Через `egrul.lookup_company(inn=defendant.inn)` проверь:
  - **Статус компании** (active / ликвидирована / реорганизация)
  - **Адрес** (для правильного места направления иска)
  - **Руководитель** (для шапки иска)

**Если статус "ликвидирована" — критично:**
Уведоми юриста: подача иска к ликвидированному юр.лицу невозможна.
Если есть правопреемник — иск направляется к нему. Если нет — иск может быть
направлен к учредителям (специальные основания).

### Шаг 3 — Досудебный порядок

Для некоторых категорий **обязателен досудебный порядок**:
- **Защита прав потребителей** — претензия не обязательна, но рекомендована
  (10/30-дневный срок ответа)
- **Договорные споры** — претензионный порядок если прописан в договоре
- **Налоговые споры** — обязательное обжалование в вышестоящий орган
- **Трудовые споры о восстановлении** — комиссия по трудовым спорам (опц.)
- **Споры о ущерб от ДТП с ОСАГО** — обращение к страховщику обязательно

Если категория требует досудебного порядка и `pre_trial_compliance=false` —
**уведоми юриста**: иск могут оставить без рассмотрения (ст. 222 ГПК).

### Шаг 4 — Расчёт цены иска и госпошлины

**Цена иска** определяется ст. 91 ГПК:
- Для денежных взысканий = сумма взыскания + проценты на дату подачи
- Для имущества = стоимость имущества (по оценке)
- Для срочных платежей = сумма платежей за 3 года вперёд
- Для договорных споров о признании права = по правилам ст. 91 ГПК

**Госпошлина** по НК РФ ст. 333.19. Используй constants registry:

  Через `constants.get_court_fees("gpk_property_claim")` получи актуальные тиры
  расчёта пошлины. Применяй формулу:

| Цена иска | Госпошлина |
|---|---|
| До 20 000 ₽ | 4%, мин. 400 ₽ |
| 20 001 - 100 000 ₽ | 800 ₽ + 3% свыше 20 000 |
| 100 001 - 200 000 ₽ | 3 200 ₽ + 2% свыше 100 000 |
| 200 001 - 1 000 000 ₽ | 5 200 ₽ + 1% свыше 200 000 |
| Свыше 1 000 000 ₽ | 13 200 ₽ + 0.5% свыше, макс. 60 000 |

Для **неимущественного иска**:
- Физлицо: 300 ₽
- Юр.лицо: 6 000 ₽

**Льготы** (ст. 333.36 НК — освобождение от пошлины):
- Истец по трудовому спору
- Истец по иску о алиментах
- Истец-потребитель при цене ≤ 1 млн ₽
- Истец-инвалид I/II группы

Если истец имеет право на льготу — указываем "истец освобождён от уплаты на
основании ст. 333.36 п. X НК РФ".

### Шаг 5 — Правовое обоснование

Подбери релевантные нормы для категории спора:

  Через `pravo.search_npa(query="...")` получи актуальные редакции:
  - **Трудовой:** ТК РФ (relevant главы)
  - **Семейный:** СК РФ
  - **Договорный:** ГК РФ Часть 2 (соответствующая глава)
  - **Возмещение вреда:** ГК РФ Глава 59
  - **Потребительский:** ЗоЗПП (2300-1)
  - **Жилищный:** ЖК РФ
  - **Наследственный:** ГК РФ Часть 3 (раздел V)

**Важно:** цитируй ТОЛЬКО ту редакцию что вернул pravo (там встроена дата
последней правки). Если в твоей training data другая редакция — игнорируй её.

### Шаг 6 — Расчёт процентов / неустойки (если применимо)

Если иск содержит требование о процентах:

  Через `cbr.calculate_penalty(principal_amount=..., days_overdue=..., rate_basis="refinancing")`
  получи актуальный расчёт по ст. 395 ГК (по ключевой ставке ЦБ).

Если иск о возмещении ущерба по ОСАГО — отдельная формула неустойки
(1% от страховой суммы за каждый день, но не более суммы возмещения).

### Шаг 7 — Список приложений (ст. 132 ГПК)

**Обязательные приложения:**
1. Документы, подтверждающие фактическое обстоятельство дела
   (договор, переписка, акт, свидетельские показания, и т.д.)
2. Документ, подтверждающий уплату госпошлины (или ходатайство о льготе)
3. Доверенность представителя (если иск через представителя)
4. Расчёт цены иска (если денежный)
5. Доказательства соблюдения досудебного порядка (где требуется)
6. Копии иска по числу ответчиков и третьих лиц

**Опционально:**
- Документы, подтверждающие имущественное положение (для алиментов)
- Заключение специалиста (для сложных случаев)
- Видео/фото доказательства (на электронных носителях)

## Degraded mode

**Если `pravo-mcp` недоступен:**
- Используй НПА из training data, **но обязательно помеч** в output:
  `⚠ Редакция нормативного акта не сверена. Перепроверьте через pravo.gov.ru
  на дату подачи иска.`

**Если `egrul-mcp` недоступен (юр.лицо):**
- Не подтверждай статус компании
- В output: `⚠ Контрагент по ЕГРЮЛ не проверен. Перепроверьте вручную через
  nalog.gov.ru перед подачей иска.`

**Если `cbr-mcp` недоступен (для расчёта процентов):**
- Используй последнюю известную ключевую ставку из constants/2026.yaml
- В output: `⚠ Ключевая ставка использована из локального кеша. Перепроверьте
  актуальное значение на cbr.ru.`

## Известные ограничения

- **Не для арбитражных дел** — есть отдельный skill `claim-draft-arbitration`
- **Не подбирает прецеденты** — для этого вызывайте `gas-pravosudie.search_similar_cases`
  отдельно (когда MCP будет готов)
- **Не учитывает региональные особенности** (например, специфика московских судов)
- **Не заполняет доверенность** — отдельный документ
- **Не проверяет срок исковой давности глубоко** — только базовая проверка
  (для сложных случаев — отдельный skill `procedural-deadlines-calc`)
