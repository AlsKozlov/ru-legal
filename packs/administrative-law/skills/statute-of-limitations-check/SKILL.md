# /statute-of-limitations-check

## Назначение

Quick calculation срока давности для конкретной admin matter. Если истёк — это
**обязательное** основание для прекращения дела.

## Workflow

### Шаг 1. Identify статью КоАП

### Шаг 2. Apply applicable срок (ст.4.5 КоАП — 2026)

#### Базовый срок

- **2 месяца** со дня совершения

#### Срок 1 год — для следующих категорий

- Налоговое законодательство и таможенное дело (НК / ТамК)
- Антимонопольное законодательство (135-ФЗ)
- Бухгалтерский учёт и финансовая отчётность
- Внешнеэкономическая деятельность
- Миграционное законодательство (ФЗ-115)
- Валютное законодательство и акты валютного регулирования
- Финансы / банковская / страхование / РЦБ
- ПДн (152-ФЗ)
- Использование атомной энергии
- Рынок ценных бумаг и инвестиционных фондов
- Использование лесного законодательства
- Природоохранное законодательство (за определёнными)
- ЖКХ (определённые составы)
- Пожарная безопасность
- Трудовое законодательство (для определённых)
- Иные (см. ст.4.5 ч.1)

#### Срок 2 года

- Защита прав потребителей
- Безопасность дорожного движения (определённые)

#### Срок 3 года

- Финансирование терроризма / экстремизма

#### Срок 6 лет

- Коррупционные правонарушения (наиболее серьёзные)

### Шаг 3. Identify дату начала

#### Default: со дня **совершения**

- Если знаем точную дату — start from there

#### Длящееся правонарушение

- Continuous violation (e.g., не уведомили РКН о изменении статуса оператора ПДн)
- Срок considered со дня **обнаружения**
- → Может быть выгодно argue что нарушение discrete не continuous

#### Особые правила

- Для admin.правонарушений обнаруженных в результате проверки — со дня **обнаружения** в акте

### Шаг 4. Calculate end date

```markdown
## Расчёт

- **Дата совершения:** [DD.MM.YYYY]
- **Применимый срок:** [2 мес / 1 год / 2 года / 3 года / 6 лет]
- **Дата истечения срока:** [DD.MM.YYYY]
- **Сегодня:** [DD.MM.YYYY]
- **Дней до истечения:** [N] / **Дней после истечения:** [N]

## Conclusion

[Срок не истёк / Срок истекает через X дней / Срок истёк X дней назад]
```

### Шаг 5. Implications

#### Если срок не истёк

- Дело рассматривается своим чередом
- Срок может пройти до вынесения постановления → all good
- Watch для preservation (если орган намеренно затягивает)

#### Если срок истекает скоро

- Strategy — затянуть процесс до истечения
- Ходатайства об отложении / о дополнительных доказательствах
- Внимание — продление срока не допускается (срок преклюзивный)

#### Если срок истёк до вынесения постановления

→ **Обязательное** прекращение дела по ст.24.5 ч.1 п.6 КоАП.

→ Use это как primary argument в возражениях / жалобе.

#### Если постановление вынесено после истечения срока

→ Постановление **незаконно** — обязательная отмена при обжаловании.

## Common pitfalls

### 🔴 Critical

- **Неправильная статья КоАП** в анализе → wrong срок
- **Confusion длящееся vs discrete** — определяющее когда начинается
- **Срок исчисления исчисление в днях / месяцах** — careful (по ст.4.8 КоАП)

### 🟡 Medium

- Срок прерывается возбуждением дела (но not paused для рассмотрения)
- Сроки давности дисквалификации — отдельные (ст.4.5 ч.4)
