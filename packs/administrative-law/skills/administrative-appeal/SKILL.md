# /administrative-appeal

## Назначение

Draft жалобы на постановление по делу об административном правонарушении.

## Pre-flight CRITICAL

### Срок (ст.30.3 КоАП)

**10 суток** с момента **получения** постановления.

⚠ **Пропуск** = постановление вступает в силу + штраф обязателен к уплате (60 дней по ст.32.2).

### Куда

| Кем вынесено | Куда обжаловать |
|--------------|------------------|
| Должн.лицом органа | Вышестоящее должн.лицо / в суд (выбор) |
| Коллегиальным органом | Вышестоящему органу / в суд |
| Судьёй мирового суда | В районный суд |
| Судьёй районного суда | В областной / приравненный суд |

### Госпошлина

**Бесплатно** для физ.лиц + ИП + ЮЛ при обжаловании.

## Workflow

### Шаг 1. Analyze постановление

```markdown
## Постановление analysis

- **№ дела:** [N]
- **Дата вынесения:** [DD.MM.YYYY]
- **Дата получения:** [DD.MM.YYYY] ← START of 10-day clock
- **Орган / должн.лицо:** [...]
- **Привлечён по статье:** [ст.X КоАП]
- **Наказание:** [штраф X / предупреждение / дисквалификация / иное]
- **Размер штрафа:** [сумма]
```

### Шаг 2. Identify основания для обжалования

#### A. Substantive

- **Состав правонарушения** не доказан
- **Отсутствие вины**
- **Малозначительность** (ст.2.9) не применена
- **Смягчающие** (ст.4.2) не учтены
- **Сроки давности** (ст.4.5) истекли

#### B. Procedural

- **Состав протокола** нарушен (ст.28.2 КоАП — обязательные реквизиты)
- **Не уведомлён** о времени / месте рассмотрения дела
- **Не присутствовал** на рассмотрении без причины (нарушает право на защиту)
- **Доказательства** получены с нарушением закона (например, без присутствия представителя)
- **Должн.лицо** не имело полномочий
- **Истекли** процессуальные сроки

#### C. Чрезмерное наказание

- Назначен максимум без отягчающих
- Не учтены смягчающие
- Нарушены критерии назначения (ст.4.1)

### Шаг 4. Подача

#### Через какие channels

- Через лицо, вынесшее постановление (forwards в вышестоящий орган / суд в 3 раб.дня)
- Напрямую в вышестоящий орган / суд
- Через Госуслуги (для некоторых категорий)
- Лично в канцелярию / почтой РФ (доказательство отправки!)

### Шаг 5. Рассмотрение

- Сроки рассмотрения: 10 раб.дней судьёй / 10 суток должн.лицом
- Может быть продлено
- Право участвовать в рассмотрении (явка / представитель)

### Шаг 6. Возможные исходы (ст.30.7)

| Исход | Что означает |
|-------|--------------|
| Оставлено без изменения | Жалоба отклонена; постановление в силе |
| Отменено + прекращено производство | Дело закрыто; штраф не платится |
| Отменено + возвращено на новое рассмотрение | Будет new рассмотрение |
| Изменено (например, снижен штраф) | Partial relief |

### Шаг 7. Дальнейшее обжалование (если не satisfied)

#### Если рассматривал районный суд (mirroвой)

→ Жалоба в районный суд — 10 дней

#### Если рассматривал районный суд (как 1st инстанция)

→ Жалоба в областной суд / приравненный — 10 дней

#### Решение вышестоящего органа

→ Жалоба в суд — 10 дней с дня получения решения

#### Кассация / надзор

Возможна экстраординарная инстанция; срок 6 мес. Гораздо узкие основания.

## Strategic considerations

### A. Selective vs. comprehensive

- **Selective** — pick 2-3 сильнейших аргументов, focus на них
- **Comprehensive** — все возможные основания

→ Recommend selective для clarity / persuasiveness; comprehensive для preserving issues для дальнейших инстанций.

### B. Outside-адв ФПА involvement

- Material штраф > 100k → recommend outside-адв
- Дисквалификация → mandatory outside-адв
- High visibility / репутационные риски → outside-адв

### C. Settlement / negotiation

В КоАП нет formal settlement. Но possible:
- Доплата штрафа в 20-дневный срок — **скидка 50%** (ст.32.2 ч.1.3 — для определённых составов)
- Withdrawal жалобы при agreement с органом — rare

## Что НЕ делает

- Не подаёт жалобу (нужна подпись + физический submission)
- Не handles court appearances — это сам / представитель
- Не replaces outside-адв ФПА для material матерей

## Why RU-unique

КоАП — РФ-specific framework. US administrative appeals — agency-specific procedures + federal courts. РФ — единый КоАП + 10-day standard window.

**Правовая основа:**
- КоАП ст.30.1-30.10 (обжалование)
- КоАП ст.30.3 (10-day срок)
- КоАП ст.30.7 (полномочия инстанции)
- КоАП ст.32.2 ч.1.3 (50% скидка)
- АПК / ГПК (если суд — соответствующая процедура)
