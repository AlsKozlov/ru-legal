# /fines-calculator

## Назначение

Calculate realistic штраф range для конкретного нарушения. Use для:
- Strategic decision (платить vs обжаловать)
- Cost / benefit анализа защиты
- Settlement negotiations

## Workflow

### Шаг 1. Find статья КоАП

### Шаг 2. Identify санкции по статье

Каждая статья КоАП имеет санкцию диапазон вид:

```
Штраф для физ.лиц: от [X] до [Y] руб
Штраф для должн.лиц: от [Z] до [W] руб
Штраф для ЮЛ: от [V] до [U] руб
+ Альтернативные санкции (предупреждение / арест / дисквалификация / ...)
```

### Шаг 3. Identify subject

```markdown
## Subject of liability

- **Тип:** [физ.лицо / должн.лицо / ИП / ЮЛ]
- **Применимая санкция:** [X-Y руб]
```

**Note:** Должн.лицо ≠ ЮЛ. Если правонарушение совершено ЮЛ, могут быть привлечены **обе** subject: должн.лицо (директор / иной ответственный) + само ЮЛ. **Different fines!**

### Шаг 4. Apply modifications

#### A. Базовый расчёт

Default — middle of range:
- Average = (min + max) / 2

#### B. Смягчающие (ст.4.2)

Если applicable:
- Раскаяние
- Добровольное сообщение о правонарушении
- Активное содействие
- Предотвращение последствий
- Беременность / малолетние дети (физ.лицо)
- Иные

→ Move к нижней границе диапазона.

→ Возможно **ниже минимума** (ст.4.1 ч.3.2):
- Для должн.лиц / ИП / ЮЛ — не менее **50%** от минимума статьи
- Apply если суд / должн.лицо найдёт исключительные обстоятельства

#### C. Отягчающие (ст.4.3)

Если applicable:
- Продолжение противоправного поведения
- Повторное совершение однородного
- Вовлечение несовершеннолетних
- Группой
- Иные

→ Move к верхней границе.

#### D. Замена штрафа предупреждением (ст.4.1.1)

Для **МСП** + first commission:
- Штраф может быть заменён **предупреждением** (no fine)
- Применимо для определённых статей (не для всех)
- Conditions:
  - Subject — МСП по реестру
  - Совершение first time
  - Нет ущерба
  - Не угрожало жизни / здоровью

#### E. Малозначительность (ст.2.9)

Если применимо:
- Лимит → **устное замечание** (no fine)
- Apply requires обоснование

#### F. 50% скидка (ст.32.2 ч.1.3)

Если уплата в **20 дней** с момента вручения постановления:
- **50% от назначенного штрафа** — для определённых статей (в основном — административные правонарушения в области дорожного движения, но также other categories)
- Verify применимость для конкретной статьи

### Шаг 5. Calculate scenarios

```markdown
## Scenario analysis

### Without modifications

- Default range: [min - max] руб
- Likely middle: [average] руб

### With смягчающими (best case)

- Apply ст.4.2 → к нижней границе: [min] руб
- Apply ст.4.1 ч.3.2 → 50% от min: [min/2] руб (если есть исключительные обстоятельства)

### With отягчающими (worst case)

- Apply ст.4.3 → к верхней: [max] руб

### С 20-day discount

- Apply ст.32.2 ч.1.3 → 50% от выписанного: [50%] руб

### Best possible — малозначительность

- Apply ст.2.9 → **устное замечание** (0 руб)

### МСП discount (если применимо)

- Apply ст.4.1.1 → **предупреждение** (0 руб) при first offense
```

### Шаг 6. Decision matrix

```markdown
## Strategic decision

### If фактический штраф low (< 50k)

- Cost defense > штраф
- Consider простая уплата с 20-day discount (если applicable)
- Defense только если material reputational impact

### If штраф material (50k - 500k)

- Defense через `/koap-violation-response` обычно cost-effective
- Probability снижения через смягчающие / процессуальные нарушения
- Outside-адв ФПА maybe

### If штраф large (> 500k) или дисквалификация

- **Mandatory** outside-адв ФПА
- Full defense + appeal strategy
- Engage immediately
```

### Шаг 7. Output

```markdown
# Fines Calculator — [matter]

## Subject + статья

- Subject: [тип]
- Article: КоАП ст.[X]
- Базовый range: [min - max] руб

## Scenario calculations

| Scenario | Probability | Сумма |
|----------|-------------|-------|
| Без modifications | 30% | [average] |
| Со смягчающими | 25% | [min] |
| С исключительными (50% от min) | 10% | [min/2] |
| С отягчающими | 15% | [max] |
| Малозначительность (ст.2.9) | 10% | 0 |
| МСП первое нарушение (ст.4.1.1) | 10% | 0 |

**Expected value:** [weighted average] руб
**Best case:** 0 руб
**Worst case:** [max] руб

## С 20-day discount (если применимо)

- If pay in 20 days: [50% of likely amount]
- Savings: [...]

## Recommended action

[Concrete: платить / defend / mixed]
```

## Common categories examples

### Ст.5.27 (Trudovaya нарушения)

- ч.1 — на ЮЛ: 30-50k
- ч.2 повторное — 50-70k
- ч.5 (зарплата) — 30-50k
- ч.6 (МРОТ) — 30-50k за каждого работника

### Ст.13.11 (152-ФЗ ПДн)

- Базовое нарушение для ЮЛ: 30-150k
- Повторное: 60-500k
- Большие массивы / sensitive: до **18 млн** (с 2025)
- Утечка: до 15 млн / 2-15% от выручки + повторное до 500М

### Ст.14.5 (Реклама)

- Для ЮЛ: 100-500k

### Ст.15.27 (115-ФЗ AML)

- Базовое: 50-500k для ЮЛ
- Повторное: 200k-1М + дисквалификация для должн.лиц

### Ст.7.30 (44-ФЗ гос.закупки)

- Различные части: 5k-300k

### Ст.20.3.3 (discreditation ВС РФ)

- 30-100k для физ.лиц; до 1М для ЮЛ; повторное — до 5М
