# /koap-violation-response

## Назначение

Killer skill — comprehensive response на протокол об административном
правонарушении. Critical для compliance + minimizing penalties.

## Pre-flight critical

### Срок реагирования

| Stage | Срок | Норма |
|-------|------|-------|
| Возражения / объяснения по протоколу | До рассмотрения дела (typically 15 раб.дней) | ст.28.2 КоАП |
| Явка на рассмотрение | По вызову | ст.29.7 |
| Обжалование постановления | **10 суток** с получения | ст.30.3 |
| Обжалование решения по жалобе | 10 суток | ст.30.9 |

⚠ **10 дней — критично**. Пропуск делает постановление final + штраф обязательным к уплате (в 60 дн).

### Категории дел (определяет кто рассматривает)

| Категория | Рассматривает |
|-----------|----------------|
| Несложные | Должностное лицо органа (issuing protocol) |
| Связанные с дисквалификацией | Судья |
| Налоговые / финансовые | Должностное лицо ФНС |
| Иные | По specific КоАП norm |

### Шаг 2. Legal analysis

#### A. Statute of limitations check (ст.4.5)

См. также `/statute-of-limitations-check`.

**Общие сроки:**

| Тип правонарушения | Срок |
|-------------------|------|
| Большинство | **2 месяца** со дня совершения |
| Длящиеся правонарушения | **2 месяца** со дня обнаружения |
| Налоговые, антимонопольные, миграционные, валютные, finансовые | **1 год** |
| ПДн (152-ФЗ), пожарная безопасность, ЖКХ, ТК — отдельные | **1 год** |
| Финансирование терроризма | **3 года** |
| Некоторые специальные | **6 лет** |

→ Если срок пропущен → **обязательное** прекращение дела (ст.24.5 ч.1 п.6).

#### B. Малозначительность (ст.2.9)

Если правонарушение **малозначительное** — суд / орган может ограничиться **устным замечанием** без наказания.

Criteria для малозначительности:
- Низкая степень общественной опасности
- Отсутствие угрозы охраняемым отношениям
- Минимальный или отсутствующий ущерб
- Признание вины + добровольное устранение

→ Argue для малозначительности если applicable.

#### C. Смягчающие обстоятельства (ст.4.2)

- Раскаяние / признание
- Добровольное сообщение о правонарушении (если не от регулятора first)
- Активное содействие установлению обстоятельств
- Предотвращение вредных последствий
- Совершение в состоянии сильного душевного волнения / при тяжёлых life circumstances
- Беременность / наличие малолетних детей (для физ.лиц)
- Первичность нарушения

→ Argument для **снижения** штрафа в пределах статьи / даже до **меньшего** минимума (ст.4.1 ч.3.2).

#### D. Отягчающие обстоятельства (ст.4.3)

- Продолжение противоправного поведения после требования прекратить
- Повторное совершение однородного правонарушения
- Вовлечение несовершеннолетних
- Совершение группой лиц
- (Иные)

→ Если применимы — argue against / minimize.

#### E. Истечение сроков альтернативно

Помимо ст.4.5:
- Истечение срока **давности привлечения** (ст.4.5) — обязательное прекращение
- Истечение срока **давности исполнения** (ст.31.9) — 2 года с вступления постановления в силу

#### F. Применимость наказания

Проверить:
- **Состав правонарушения** действительно соответствует диспозиции?
- **Subject** правильно определён (физ.лицо vs ИП vs ЮЛ vs должн.лицо)?
- **Form вины** (умысел / неосторожность) requires consideration?
- **Сumulative санкции** запрещены за тот же случай

### Шаг 4. Strategy options

#### Option A: Defend полностью (нет правонарушения)

- Аргументы по существу
- Доказательства отсутствия вины / состава
- Recommend если strong basis

#### Option B: Mitigate (правонарушение есть, но fine reducible)

- Аргументы по смягчающим
- Признание + раскаяние
- Минимальный штраф по статье

#### Option C: Прекратить за давностью / малозначительностью

- Если applicable

#### Option D: Платить fine

- Если защита weak + штраф minor + cost защиты > штраф
- "Скидка 50%" если уплачен в **20 дней** (ст.32.2 ч.1.3) — но **применима НЕ ко всем составам**, а только определённому перечню (в основном **глава 12 КоАП** — нарушения ПДД; также отдельные составы в гл.6, 11, 14). **Verify** конкретную статью КоАП — для большинства «предпринимательских», налоговых, ПДн, ЖКХ-составов скидка **не действует**

### Шаг 6. Участие в рассмотрении

```markdown
## Pre-meeting prep

- [ ] Возражения поданы заранее (если возможно)
- [ ] Документы готовы для презентации
- [ ] Outside-адв запланирован к участию (recommended для material матерей)
- [ ] Свидетели уведомлены / готовы

## В заседании

- Объявить позицию по существу
- Представить доказательства
- Argue смягчающие / отсутствие отягчающих
- Argue малозначительность / истечение сроков
- Получить копию постановления
```

### Шаг 7. После рассмотрения — постановление

#### Если благоприятное

- Прекращение дела → конец
- Штраф минимальный → платить (в 60 дн)

#### Если неблагоприятное → обжалование

→ See `/administrative-appeal`. Strict 10-day clock.

## Что НЕ делает

- Не представляет в заседании (нужен живой adv / самостоятельно)
- Не платит штраф (это бухгалтерия)
- Не handles уголовные дела (это outside-адв ФПА)

## Why RU-unique

US administrative law — federal agencies (APA, agency-specific procedures) + state. РФ — единый КоАП + 248-ФЗ + sectoral. Strict 10-day window — РФ specific.

**Правовая основа:**
- КоАП РФ (полностью, особ. ст.4, 28, 29, 30)
- 248-ФЗ «О государственном контроле»
- Постановления Правительства / Министерств — sectoral
