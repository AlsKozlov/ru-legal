# /cease-desist

## Назначение

Draft досудебной претензии по IP нарушению. Cross-references:
- `/infringement-triage` — анализ нарушения (выполнить first)
- `litigation/pretensia-draft` — общий шаблон претензии (для non-IP)
- После C&D — `litigation/matter-intake` если переходим в суд

## Pre-flight

- **`/infringement-triage`** выполнен — нарушение confirmed
- **Доказательственная база** собрана (notarial fixation если онлайн)
- **Materiality** оценена
- **Strategy** определена — это первый шаг к litigation или standalone deterrent?

## Workflow

### Шаг 1. Compile основу претензии

```markdown
## Структура претензии

1. Шапка (кому, от кого)
2. Описание нашего права (что и когда зарегистрировано)
3. Описание нарушения (что делает infringer)
4. Правовое обоснование (ст. ГК)
5. Требования (cease + compensation + другие)
6. Срок (30 дней — стандарт для досудебки по ст.4 ч.5 АПК)
7. Указание на последующие действия если игнор
8. Приложения
9. Подпись + дата
```

## 2. Описание нарушения

[Полное наименование infringer] (далее — Нарушитель) совершает следующие действия,
нарушающие исключительные права Правообладателя:

### Конкретные действия

1. **[DD.MM.YYYY]** — Нарушитель производит / продаёт / рекламирует [товары / услуги]
   с использованием обозначения **[нарушающее обозначение]**, сходного до степени
   смешения с товарным знаком **[№ NNNNNN]** Правообладателя, в отношении
   однородных товаров [классы МКТУ].

2. **[DD.MM.YYYY]** — Нарушитель размещает на сайте [URL] / маркетплейсе
   [название] предложение к продаже товара "**[название]**", содержащего
   признаки запатентованного изобретения по патенту **№ NNNNNN**
   Правообладателя.

3. **[DD.MM.YYYY]** — иное

### Доказательства нарушения

- Скриншоты сайта [URL] от [DD.MM.YYYY], нотариально удостоверённые
  нотариусом [ФИО] (реестр № X) — приложение 2
- Контрольная закупка от [DD.MM.YYYY] — кассовый чек, образец товара —
  приложение 3
- Свидетельские показания: [ФИО] — приложение 4

## 3. Правовое обоснование

В соответствии со статьями [применимые ст. ГК — ст.1229 + 1484/1358/1270/1467]
Гражданского кодекса РФ:

- Правообладатель имеет исключительное право использовать [объект IP] любым не
  противоречащим закону способом.
- Использование указанного объекта без согласия правообладателя является
  нарушением исключительного права.

Нарушение исключительных прав влечёт ответственность по статьям **1252, [1515 /
1301 / 1311 / 1334 / 1472]** ГК РФ:
- Пресечение нарушения
- Компенсация в размере **от 10,000 до 5,000,000 руб** (на усмотрение суда) —
  для большинства типов IP
- ИЛИ компенсация в размере **двукратной стоимости** контрафактных товаров /
  стоимости использования IP

## 4. Требования

В связи с изложенным **ТРЕБУЕМ** в **тридцатидневный срок** с момента получения
настоящей претензии:

1. **Прекратить** все действия, нарушающие наши исключительные права на
   [объект IP], а именно:
   - Прекратить продажу / производство / рекламу контрафактной продукции
   - Удалить / уничтожить все имеющиеся образцы контрафактной продукции
   - Удалить с сайта [URL] / маркетплейса [...] все упоминания / изображения
     спорного обозначения
   - [иные конкретные требования]

2. **Выплатить компенсацию** в размере **[сумма] руб** в качестве возмещения
   причинённых нарушением убытков (либо в качестве заявленной компенсации по
   ст.[1515/1301/иной] ГК РФ) на расчётный счёт:
   - Получатель: [полное наименование]
   - ИНН/КПП: [...]
   - Расчётный счёт: [...]
   - Банк: [...]
   - БИК: [...]
   - Корр. счёт: [...]
   - Назначение: "Компенсация за нарушение исключительных прав на [объект IP]
     по претензии от DD.MM.YYYY".

3. **Предоставить письменное подтверждение** выполнения требований с
   доказательствами (фото удалённых страниц, документы об уничтожении товаров,
   копия платёжного поручения о выплате компенсации).

## 5. Последствия неисполнения

В случае невыполнения данных требований в установленный срок Правообладатель
будет вынужден обратиться в суд с иском о:

- Пресечении нарушения исключительных прав
- Взыскании компенсации в размере **до 5,000,000 руб** (либо в двукратном
  размере стоимости контрафактных товаров)
- Изъятии и уничтожении контрафактных товаров
- Публикации решения суда о нарушении (ст.1252 ч.5 ГК)
- Взыскании судебных расходов (в т.ч. на оплату услуг адвоката)

Также возможно обращение к правоохранительным органам по факту нарушения
[статья 146 / 147 / 180 УК РФ — если applicable по masштабу].

### Шаг 3. Расчёт компенсации (для requirement #2)

#### Methods для justify suma

**Method A: Statutory ranges (ст.1515 / 1301 ч.4)**

- 10,000 - 5,000,000 руб на усмотрение суда
- Часто use в претензии: 100,000 - 1,000,000 (даёт реалистичный bargaining point)

**Method B: 2× стоимость контрафактных товаров**

- Применимо когда materiality конкретна: продажа N единиц по цене Y → compensation = 2 × N × Y
- Лучше с conкретного infringement evidence

**Method C: 2× стоимость использования IP**

- Применимо если есть market rate license royalty
- Compensation = 2 × (royalty × period × revenue)

**Method D: Убытки** (реальный ущерб + упущенная выгода)

- Сложнее доказать
- Reserved для очень больших matters

#### Strategic suma в претензии

Часто claim сумму на 30-50% выше realistic recoverable, чтобы оставить room для negotiation. Если infringer pays % - good outcome без litigation cost.

### Шаг 4. Submission

#### Способы

- **Почтой РФ** заказным письмом с описью и уведомлением о вручении (стандарт — это start of 30-day clock)
- **Лично с проставлением штампа** в приёмной (для near locations)
- **Email** с проставлением подписи / удостоверением — допустимо, но почта надёжнее

**Critical:** preserve **proof of dispatch + delivery** — это начало 30-дневного срока досудебки.

### Шаг 5. Tracking response

#### Возможные исходы

| Исход | Дальше |
|-------|--------|
| Compliance полная (cease + payment) | Закрыть matter; document outcome |
| Compliance частичная (cease но не payment / возражения по сумме) | Negotiate settlement / proceed to litigation если сильная позиция |
| Negotiation request | Engage outside-адв ФПА для negotiation |
| Counter-claim / defense | Re-assess strength позиции; consider settlement |
| Ignore / no response в 30 дней | Litigation — `litigation/matter-intake` + СИП |
| Acknowledgement без compliance | Same — proceed to litigation |

### Шаг 6. Strategic considerations

#### A. Tone

- Formal, professional
- No personal attacks
- No exaggeration (документация faktической базы)
- Confidence — на основании рreal evidence

#### B. Multiple infringers — batch enforcement

Если есть множество infringers (например, копии в маркетплейсах) — можно send
batch C&D через outside-адв одним пакетом + automated tracking.

#### C. Press release / public

Public C&D — иногда strategic (deter others) но carries reputational risk + может
быть seen as agressive. Use с осторожностью.

#### D. Cross-border нарушения

Если infringer вне РФ — РФ C&D limited utility. Need:
- C&D in jurisdiction infringer (требует local counsel)
- РФ судебное решение для enforcement в их jurisdiction (recognition obtain)

#### E. Online platforms (takedown отдельно)

Параллельно с C&D к нарушителю — можно send takedown notice платформе (см. `/takedown`).
Часто платформа быстрее реагирует чем сам нарушитель.
