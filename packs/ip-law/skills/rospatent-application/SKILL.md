# /rospatent-application

## Назначение

Подготовка к подаче и tracking заявки в Роспатент. **Note:** для иностранных
заявителей **обязателен патентный поверенный** (ст.1247 ГК). Для РФ residents —
not formally required, но highly recommended.

## Pre-flight

- **Объект для регистрации** identified
- **`/clearance`** done (для ТЗ) или patentability analysis (для патентов)
- **Бюджет** известен (госпошлины + услуги поверенного)
- **Документация** готова (название, изображение, формула, описание, и т.д.)

## Типы заявок + базовые spec (2026)

### A. Товарный знак

| Этап | Срок | Госпошлина (2026) |
|------|------|-------------------|
| Подача заявки | n/a | 35,000 + 11,500 за каждый класс свыше 1 |
| Формальная экспертиза | 2 мес | - |
| Экспертиза по существу | 12 мес (часто продляется до 18 мес) | - |
| Регистрация ТЗ | n/a | 16,000 + 1,000 за каждый класс свыше 1 |
| Свидетельство | n/a | (in рамках регистрационной) |
| Срок действия | 10 лет | - |
| Продление каждые 10 лет | n/a | 20,000 + 1,000 за каждый класс |

### B. Изобретение

| Этап | Срок | Госпошлина |
|------|------|------------|
| Подача | n/a | 4,700 (1 ind.claim) + 700 за каждый ind.claim сверх 1 + 320 за каждый страницу описания свыше 25 |
| Формальная экспертиза | 2 мес | - |
| Экспертиза по существу | до 12 мес после требования | 11,800 + 4,700 за каждый ind.claim |
| Регистрация | n/a | 4,700 |
| Срок действия | 20 лет с подачи | Annual fees with 3-го года |

### C. Полезная модель

- Срок защиты: max 10 лет
- Госпошлины — ниже изобретения, но similar structure
- Только для устройств (не способов / веществ)
- Без проверки изобретательского уровня — quick procedure

### D. Промышленный образец

- Срок защиты: 5 лет + 4 продления по 5 лет = max 25 лет
- Госпошлины — similar

### E. Программа для ЭВМ / База данных

| Этап | Срок | Госпошлина |
|------|------|------------|
| Подача | n/a | 4,500 (если ЮЛ) / 3,000 (если физ.лицо) |
| Регистрация | 2 мес | - |
| Свидетельство | n/a | (in рамках регистрационной) |
| Срок защиты | пожизненно для ПО + 70 лет автору после смерти; для БД — 15 лет с создания | - |

**Critical:** Регистрация ПО / БД **не обязательна** — авторские права возникают
автоматически с создания. Но регистрация:
- Дает доказательную силу авторства + даты создания
- Required для включения в реестр Минцифры (IT-аккредитация)
- Дает право использовать символ ©

### F. НМПТ (Наименование места происхождения товара)

- Специфическая категория (ст.1516 ГК)
- Применимо: "Тульский пряник", "Хохлома", "Гжель", "Дагестанский коньяк", и т.п.
- Требует особенностей качества связанные с географическим origin

### Шаг 2. Collect required documents

#### Для ТЗ

- [ ] **Заявление о регистрации** ТЗ (форма по приказу Минэкономразвития)
- [ ] **Изображение знака** (для словесного — текст; для изобразительного — high-res)

- [ ] **Описание знака** (опционально, но полезно)
- [ ] **Документ об уплате госпошлины**
- [ ] **Доверенность** на патентного поверенного (если применимо)

#### Для изобретения

- [ ] **Заявление о выдаче патента**
- [ ] **Формула изобретения** (заглавный пункт + зависимые)
- [ ] **Описание изобретения** (структурированное: область техники, известный уровень, существо, краткое описание чертежей, осуществление, преимущества)
- [ ] **Чертежи** (если применимо)
- [ ] **Реферат**
- [ ] **Документ об уплате госпошлины**
- [ ] **Доверенность**

#### Для ПО / БД

- [ ] **Заявление о регистрации**
- [ ] **Депонируемые материалы:**
  - Reference text (фрагмент кода / описание БД)
  - Реферат
  - Сведения об авторе (с заверением о выплате вознаграждения если служ.)
- [ ] **Документ об уплате госпошлины**

### Шаг 3. Submission channels

| Channel | Best for |
|---------|----------|
| **ЛК ФИПС** (электронно) | Recommended — fastest, паперless |
| **Лично** в приёмной ФИПС в Москве | Если в Москве + сложная заявка |
| **По почте** | Last resort |

### Шаг 4. Tracking expertise

После подачи:

#### Формальная экспертиза (2 мес)

Проверка:
- Все required документы есть
- Госпошлина уплачена
- Format correct
- Сведения about заявителя verified

#### Запрос экспертизы по существу

После формальной — может быть **запрос (Office Action)** от ФИПС:
- Если есть основания для отказа
- Срок ответа: **6 мес** (можно продлить до 12 мес с дополнительной пошлиной)
- Должен submitted argued response + corrections если необходимо

### Шаг 5. Office Action response

#### Common основания запроса (для ТЗ)

- Confusing similarity с existing TM
- Описательность
- Введение в заблуждение
- Иные absolute / relative grounds (ст.1483 ГК)

#### Common основания (для патента)

- Не отвечает условию новизны
- Не отвечает условию изобретательского уровня
- Не отвечает промышленной применимости
- Формула изобретения не четкая

#### Strategy в ответе

- Аргументы по существу + при необходимости amendments
- Если confusing similarity claim → "consent letter" от owner cited mark
- Если новизна challenged → demonstrate distinguishing features
- Если изобретательский уровень → technical advantages + economic benefits

### Шаг 6. Регистрация после allowance

- Уплата регистрационной пошлины
- Получение свидетельства / патента
- Запись в государственном реестре ФИПС

### Шаг 7. Post-filing maintenance

- Patents: annual fees with 3rd year
- TM: maintenance до renewal at 10 years
- ПО / БД: lifetime для автора + 70 лет

## РФ-specific considerations

### A. Электронное взаимодействие с ФИПС

- ЛК ФИПС позволяет полностью paperless процесс
- Ускоренная переписка через ЛК
- Recommended использовать quote вместо paper

### B. EAEU единый ТЗ

С 2021 — possible подать единую заявку, действительную в 5 странах (РФ, Беларусь,
Казахстан, Армения, Киргизия):
- Через EAPO (Eurasian Patent Office)
- Госпошлина — combined
- Прибыл cheaper than 5 separate national filings

### C. Мадридская система (international ТЗ)

- Базовая национальная регистрация в РФ — prerequisite
- Затем подать international application через WIPO
- Designate countries
- Госпошлины — variable

### D. PCT (international patents)

- Подача national / regional + filing PCT в течение 12 мес
- 30/31 month deadline для national phases в Designated states
- Provides time для оценки commercial viability

### E. Sanctions effects post-2022

- Some countries restricted USPTO / EPO accept РФ filings
- РФ → other countries — варьируется (Беларусь, СНГ, Китай, Индия — fully operational)
- US / EU filings — practically more difficult; engage in-country counsel
- Sanctions package periodically updates — check current state

## Что НЕ делает

- Не actually подаёт через ЛК ФИПС (нужен патентный поверенный или сам заявитель)
- Не handles refusals appeal — это Палата по патентным спорам Роспатента + СИП
- Не handles cross-border filing logistics — это international patent attorneys

**Правовая основа:**
- Часть 4 ГК РФ (особенно ст.1245-1247, 1262, 1350, 1351, 1352, 1477, 1492, 1516)
- ПП РФ № 793 о патентных пошлинах
- Положение о патентных пошлинах № 941
- Постановления Минэкономразвития о формах заявок
- ст.1247 ГК — патентный поверенный для иностранных заявителей
