# /infringement-triage

## Назначение

Quick assessment — действительно ли есть нарушение наших IP rights, какой
materiality, какая стратегия (cease & desist, litigation, ignore).

## Pre-flight

- **Какой именно IP затронут:**
  - Товарный знак (ст.1484, защита по ст.1515)
  - Патент на изобретение / полезную модель / промобразец (ст.1358, защита по ст.1252)
  - Авторские (ст.1255+, защита по ст.1301)
  - ПО / БД (ст.1261, 1334)
  - Ноу-хау (ст.1465, защита по ст.1472)
- **Доказательства** нарушения собраны
- **Material assessment** (стоимость потенциальной потери, repuт.risk)

## Workflow

### Шаг 1. Identify IP at issue + наша protection

```markdown
## Наш IP

- **Тип:** [...]
- **Регистрационный номер (если зарегистрирован):** [...]
- **Дата приоритета / регистрации:** [...]
- **Срок действия (до):** [...]
- **Объём защиты:** [для ТЗ — классы МКТУ; для патента — claims; для авторских — конкретный объект]
- **Territory:** [РФ / EAEU / Мадрид international / иное]
```

### Шаг 2. Identify alleged infringer + действия

```markdown
## Alleged infringer

- **Имя / название:** [...]
- **ИНН / ОГРН (если ЮЛ РФ):** [...]
- **Адрес:** [...]
- **Type:** [физ.лицо / ИП / ЮЛ РФ / иностранец]

## Alleged actions

- **Что делает:** [конкретные actions — продажа / реклама / производство / импорт / online платформа]
- **Где:** [магазин / маркетплейс / сайт / соц.сети]
- **Период:** [...]
- **Масштаб (estimate):** [количество единиц / охват аудитории / доход]
```

### Шаг 3. Anaylze actual infringement

#### Для ТЗ (ст.1484, 1515 ГК)

**Нарушение** = использование без согласия:
- Сходного обозначения на товарах однородных классов МКТУ
- На сайте, в рекламе, на упаковке
- В фирменном наименовании / коммерческом обозначении
- В доменном имени

**Confusing similarity test** — см. analysis в `/clearance`.

#### Для патента (ст.1358 ГК)

**Нарушение** = осуществление действий с использованием **каждого признака** независимого пункта формулы изобретения / полезной модели:
- Изготовление
- Применение
- Предложение к продаже
- Продажа, иное введение в гражданский оборот
- Импорт в РФ

Test: **claim-chart** анализ (см. `/fto-triage`).

#### Для авторских (ст.1255+ ГК)

**Нарушение** = воспроизведение, распространение, переработка, доведение до всеобщего сведения без согласия:
- Прямое копирование
- Производные произведения (переработка)
- Использование без указания авторства / искажение

Test:
- Объект защищён? (автор identified, объективная форма)
- Совпадение / substantial similarity?
- Доступ нарушителя к нашему произведению?

#### Для ПО / БД (ст.1261, 1334 ГК)

ПО — приравнено к литературным произведениям. БД — особая защита.

**Нарушение:** копирование кода, переработка, продажа, распространение, использование без лицензии.

#### Для ноу-хау (ст.1465-1472 ГК)

**Нарушение** = неправомерное получение / разглашение / использование секрета производства при условии что:
- В отношении секрета установлен **режим коммерческой тайны** (98-ФЗ)
- Лицо знало о confidentиальности

### Шаг 4. Defenses to consider

Alleged infringer может иметь защиту:

| Defense | Когда работает |
|---------|----------------|
| **Исчерпание права** (ст.1487 ГК) | Товар был введён в оборот с согласия правообладателя (но post-2022 параллельный импорт — separate question) |
| **Свободное использование в личных целях** (ст.1273) | Только некоммерческие цели для физ.лиц |
| **Использование в научных / культурных целях** (ст.1274-1280) | Limited |
| **Pre-existing use** (для патентов — ст.1361) | Использовали до приоритетной даты |
| **Patent invalidity** | Patent может быть challenged |
| **Принудительная лицензия** (ст.1239, 1362) | Для определённых случаев |
| **Параллельный импорт** (ПП РФ 506/2022) | Для определённых categories товаров |

### Шаг 5. Materiality assessment

```markdown
## Materiality

- **Estimated damages:** [сумма]
  - Lost sales: [сумма]
  - Reputation damage: [сумма / qualitative]
  - Royalty equivalent (если бы licensed): [сумма]

- **Compensation under ст.1252 / 1515:**
  - От 10,000 до 5,000,000 руб (по усмотрению суда)
  - ИЛИ 2× стоимость контрафактных товаров / товаров на которых нарушение
  - ИЛИ 2× стоимость использования IP (для ПО / иных)

- **Threshold для action:**
  - Critical (> 1М) → immediate aggressive action
  - Material (100k - 1М) → cease & desist + possible litigation
  - Minor (< 100k) → cease & desist only / ignore / batch enforcement
```

### Шаг 6. Доказательственная база

#### Что нужно зафиксировать

**Для online нарушения:**
- Скриншоты с дата+время (best — notarized через нотариальное удостоверение online evidence)
- URL + копия page
- Wayback Machine snapshots
- Свидетельства о доступности

**Для офлайн (товары):**
- Контрольная закупка (с фиксацией покупки — кассовый чек, фото, видео)
- Образец товара (preserved для экспертизы)
- Реклама / документы продавца

**Свидетельства правообладания:**
- Свидетельство о регистрации ТЗ / патента (полная копия)
- Выписка из реестра Роспатент актуальная
- Договоры авторские (для авторских произведений)

### Шаг 7. Strategy options

#### Option A: Cease & desist letter (досудебная претензия)

→ Use `/cease-desist`

- Обязательна по ст.1252 ГК + АПК ст.4 ч.5 для подсудности СИП — 30 дней досудебка
- Most common starting point

#### Option B: Litigation в СИП

- Для всех ТЗ + патентных споров — СИП первой инстанции (или АС субъекта в зависимости от характера, но СИП с 2013 — основная)
- Для авторских — АС субъекта по месту ответчика (или СИП)
- Для ПО / БД — АС субъекта
- Требования: пресечение + компенсация + публикация решения + изъятие контрафактных товаров
- Срок рассмотрения — 3-9 мес типично

#### Option C: Антипиратский закон (для online)

- Мосгорсуд (Москва) — exclusive jurisdiction для блокировки сайтов с пиратским контентом
- Применимо для авторских и смежных прав
- Срок — 15 дней блокировка по предварительной мере; финальная блокировка после суда
- Не для ТЗ / патентов

#### Option D: Уголовное (ст.146-147 УК)

- Для авторских / смежных нарушений с крупным ущербом (> 100k)
- Для ТЗ — ст.180 УК (незаконное использование средств индивидуализации, если ущерб > 250k)
- Engage outside-адв ФПА + обращение в правоохрану

#### Option E: Жалоба в Роспатент / ФАС

- Роспатент Палата по патентным спорам — для оспаривания регистрации (cancellation)
- ФАС — для недобросовестной конкуренции (ст.14 ФЗ-135), включая неправомерное использование IP

#### Option F: Ignore

- Минор нарушения с low materiality
- High litigation cost vs benefits
- Risk negative precedent
- Document для future batch enforcement
