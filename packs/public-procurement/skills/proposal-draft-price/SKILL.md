# /proposal-draft-price

## Назначение

Determine оптимальную bid цену для участия в тендере. Balance между:
- Желанием win (нижнее давление)
- Antidemping triggers (нижний porog для doubling security)
- Финансовой жизнеспособностью (минимальная маржа для execution)
- Стратегическим placement (часто won't be lowest)

## Pre-flight

- НМЦК известен из документации
- Тип определения поставщика (аукцион / конкурс с критериями)
- Финансовое моделирование выполнено (наша себестоимость)
- `/tender-eligibility-check` + `/tender-documentation-review` пройдены

## Workflow

### Шаг 1. Financial baseline

```markdown
## Cost analysis

- **Себестоимость для нас** (в рамках scope ТЗ):
  - Прямые затраты: [сумма]
  - Накладные: [сумма]
  - Налоги (НДС, налог на прибыль с маржи): [сумма]
  - Cost обеспечения (банк.гарантия — премия / альтернативно денежные средства lockup): [сумма]
  - **Total cost: [сумма]**

- **Target margin для нас:**
  - Minimum acceptable: [%]
  - Target: [%]
  - Maximum (если low competition): [%]

- **Break-even цена:** [сумма]
- **Target bid:** [сумма] = себестоимость × (1 + target margin / 100)
```

### Шаг 2. Market analysis (если данные доступны)

```markdown
|--------|------|-------------|-----------|
| Tender A | 10М | 8.5М | 15% |

| ... | | | |

## Конкуренция

- Сколько обычно bidders в нашей нише: [N]

- Их типичная стратегия: [агрессивный demпинг / стандартный / медленный response]
```

### Шаг 3. Antidemping analysis

Critical threshold — **25% ниже НМЦК** (ст.37 ФЗ-44).

```markdown
## Antidemping zone

- НМЦК: [сумма]
- 25% порог: [сумма × 0.75] — bids ниже этой суммы trigger antidemping

### Если bid > 75% от НМЦК (нормальная зона)

- Стандартное обеспечение исполнения (5-30%)
- Без обоснования цены

### Если bid 50-75% от НМЦК (антидемпинг)

Опция A — удвоенное обеспечение:
- Стандартное × 2 = [сумма]
- Дополнительный финансовый burden

Опция B — обоснование цены (ст.37 ч.2):
- Подтверждающие документы что мы можем выполнить за эту цену
- Например: договоры на закупку сырья по low ценам; собственное производство; опыт оптимизации; и т.п.
- Если documents убедительны — стандартное обеспечение

### Если bid < 50% НМЦК (very deep demпинг)

- Antidemping + обязательное обоснование
- Заказчик с осторожностью; risk отказа в подписании контракта если не убеждает

## Recommendation

[Какая стратегия лучше учитывая наши финансы + competition]
```

### Шаг 4. Strategic bid placement

#### Pattern A: Win at all costs

- Bid максимально низко (включая антидемпинг)
- Готовы убытить контракт для опыта / market presence
- Risk: антидемпинг + потенциальные убытки + repuт.risk если не sustain quality

#### Pattern B: Optimal margin

- Bid слегка ниже expected median (по нашим market data)
- Сохраняет хороший margin
- Probability win: 30-50% если конкуренция типичная

#### Pattern C: High margin, low probability

- Bid близко к НМЦК (на 5-10% ниже)
- Margin высокий, но Probability win ~10-20%
- Имеет sens для бутиковых тендеров с unique requirements

#### Pattern D: Strategic non-participation (No bid)

- Если все возможные bid позиции не достигают minimum margin
- Better to keep resources for другие тендеры
- Не considered "loss"; иногда правильнее не bid'ать

### Шаг 5. Draft price part

#### Template: ценовое предложение (для аукциона — динамика на торгах)

Для аукциона — цена определяется в ходе торгов. Заявка содержит **только согласие**.
Snижение цены происходит в реальном времени на ЭТП (шаг 0.5% от НМЦК).

Так что drafting цены — больше strategy planning.

```markdown
# ЦЕНОВОЕ ПРЕДЛОЖЕНИЕ
# участника закупки № [номер]
# [Наименование участника, ИНН]

## Общая цена контракта

**Предлагаемая цена:** [сумма прописью] руб ([цифрами]) с НДС [сумма].

## Разбивка по позициям (если применимо)
```

### Шаг 6. Если будет antidemping — обоснование

#### Template: Обоснование низкой цены

```markdown
# ОБОСНОВАНИЕ ЦЕНЫ
# (в порядке ст.37 ФЗ-44)

В соответствии со ст.37 ч.2 ФЗ-44 представляем обоснование предлагаемой цены
в сумме [сумма], которая ниже НМЦК [сумма] на [%].

## Структура цены

[Detailed breakdown of how мы достигаем такую цену]

### 1. Источники низкой себестоимости

- **Прямые закупки у производителя** (без посредников):
  - Договор с [Производитель] № X от DD.MM.YYYY (приложение 1)
  - Цена закупки: [сумма] / ед.
  - Сравнение с рыночной (через прайс другого поставщика): [сумма], экономия [%]

- **Собственное производство ключевых компонентов:**
  - Описание производства
  - Сертификаты на собственную продукцию

- **Оптимизация логистики:**
  - Своя транспортная служба
  - Маршруты до объекта заказчика
  - Расчёт logистики

### 2. Подтверждение возможности исполнения

- Опыт в исполнении аналогичных контрактов (приложение 2 — список)
- Финансовые ресурсы для авансирования (приложение 3 — банк.выписка)
- Кадровые ресурсы (приложение 4 — штат)

### 3. Подтверждение качества

- ISO 9001:2015 сертификат
- Контроль качества на каждом этапе
- Гарантийные обязательства (полные)

## Заверение

Подтверждаем что предлагаемая цена позволит исполнить контракт в полном объёме
с соблюдением всех условий ТЗ.

[Подпись + дата]
```
