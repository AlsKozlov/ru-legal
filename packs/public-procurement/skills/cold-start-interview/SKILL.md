# /cold-start-interview (public-procurement)

## Workflow

### Шаг 1. Тип компании + опыт

- ИП / ООО / АО
- МСП (Реестр субъектов МСП) — Y/N → даёт льготы по 44-ФЗ:
  - Заказчик обязан размещать определённую долю (минимум 25%) среди МСП
  - Сниженное обеспечение
  - Преимущественные условия
- Опыт работы с гос.заказчиками — новички / опытные

### Шаг 2. Сферы участия

- ОКВЭДы основные + дополнительные
- Категории товаров / работ / услуг
- География
- Импортные товары — Y/N (post-2022 — ограничения на некоторые)

### Шаг 3. ЭТП аккредитации

С 2018 — единая аккредитация в ЕИС → автоматически на 9 ЭТП:
- Сбербанк-АСТ
- РТС-тендер
- ЕЭТП (B2B-Center)
- ETP-MICEX
- Российский аукционный дом
- АГЗРТ
- Газпромбанк
- НЕЦПТ (на 2024 — рассматривается)
- ТЭК-Торг (для 223-ФЗ закупок Роснефти/Газпрома)

Check: аккредитация активная? Сроки действия?

### Шаг 4. Финансовые возможности

- Обеспечение заявки — размер (1-5% начальной цены контракта)
- Обеспечение исполнения — размер (5-30%, плюс антидемпинговые добавки)
- Источник: банковская гарантия (стандарт) / денежные средства
- Банки-партнёры с опытом гос.заказа
