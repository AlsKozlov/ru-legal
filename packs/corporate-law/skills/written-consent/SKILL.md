# /written-consent

## Назначение

Help draft proposal + bulletins + протокол для заочного голосования (опросным
путём) участников ООО / акционеров АО.

**В РФ это fundamentally OTHER от US Delaware "written consent in lieu of meeting":**

| Аспект | US Delaware | РФ ООО/АО |
|--------|-------------|-----------|
| Trigger | All-or-nothing — required unanimous (если не предусмотрено иное) | Большинством голосов (как и на ОСУ) |
| Form | Подписание единого документа всеми участниками | Заочный опрос — каждый голосует отдельно, результаты подсчитываются |
| Какие вопросы | All matters (если LLC agreement allow) | **Ограничения** — ст.50 ФЗ-208 запрещает заочно решать СД/ревизионную комиссию/реорганизацию/ликвидацию |
| Документация | Consent резолюция | Бюллетень для голосования + протокол |
| Нотариальное удостоверение | Не требуется | **Обязательно для ООО ОСУ** (ст.67.1 ГК — для очных и заочных) **кроме случаев если устав предусматривает иное и есть единогласие** |

## Pre-flight critical checks

### A. Допустимость заочной формы для конкретного вопроса

#### Для АО (ст.50 ФЗ-208)

**Нельзя** решать заочно:
- Избрание Совета директоров
- Избрание Ревизионной комиссии / ревизора
- Утверждение аудитора
- Утверждение годового отчёта, годовой бух.отчётности
- Распределение прибыли (выплата дивидендов по итогам года)
- Реорганизация
- Ликвидация

**Можно** решать заочно (если устав или решение СД не запрещает):
- Все остальные вопросы (но typically — только конкретные сделки, изменения условий)

#### Для ООО (ст.38 ФЗ-14)

- **По умолчанию** заочное голосование возможно по всем вопросам, кроме утверждения годовой бух.отчётности
- **Уставом** могут быть установлены additional ограничения

### B. Нотариальное удостоверение (ст.67.1 ГК с 2014)

**Для ООО ОСУ — обязательно** ст.67.1 ГК, **если уставом не предусмотрен альтернативный способ** и решение не принято единогласно.

**Альтернативные способы:**
- Подписание всеми участниками
- Иной способ, предусмотренный уставом (например — аудио/видеозапись с подписью каждого)

**Для АО:**
- ОСА — удостоверяется регистратором / нотариусом / иным предусмотренным способом
- Заочное — регистратор фактически принимает бюллетени, удостоверение через него

**Common mistake:** drafted протокол без нотариального удостоверения → решения недействительны.

### C. Срок уведомления (для очных и заочных собраний)

- ООО: **не менее 30 дней** до даты (ст.36 ФЗ-14)
- АО непубличное: **не менее 21 дня** (ст.52 ФЗ-208)
- ПАО: **не менее 30 дней** (ст.52 ФЗ-208)

Уведомление включает: повестка дня, материалы для ознакомления, для заочного — бюллетень + срок сбора.

## Workflow

### Шаг 1. Identify вопрос + допустимость заочной формы

```markdown
## Вопрос для голосования

[Текст вопроса формальный]

## Допустимость заочной формы — analysis

- Тип организации: [ООО / АО / ПАО]
- Категория вопроса: [...]
- Запрет на заочное? [Y/N + reference к ст.50 ФЗ-208 / уставу / ст.38 ФЗ-14]

**Conclusion:** заочное голосование [возможно / не возможно].
```

Если **не возможно** — STOP, recommend очное собрание + offer `/corporate-meeting-prep`.

### Шаг 2. Draft бюллетень для голосования

```markdown
# БЮЛЛЕТЕНЬ ДЛЯ ГОЛОСОВАНИЯ

**[Организационно-правовая форма + наименование]**
**Заочное голосование от [DD.MM.YYYY] — [DD.MM.YYYY]**

## Сведения о голосующем

- **ФИО (для физ.лица) / Наименование (для юр.лица):**
- **Доля в уставном капитале (для ООО) / Кол-во голосующих акций (для АО):** [% / число]
- **Документ, подтверждающий полномочия (для представителя):**

## Повестка дня

### Вопрос 1: [формулировка]

[Развёрнутая формулировка, как точно она будет в протоколе]

**Варианты голосования:**

| ☐ ЗА | ☐ ПРОТИВ | ☐ ВОЗДЕРЖАЛСЯ |
|------|----------|-----------------|

### Вопрос 2: [...]

[То же]

## Подпись

Дата заполнения бюллетеня: [DD.MM.YYYY]

Подпись: ________________ / [ФИО / для юр.лица — должность + ФИО] /

**ВАЖНО:** Бюллетень, не содержащий подписи / содержащий правки / без отметки в одной из клеток "ЗА/ПРОТИВ/ВОЗДЕРЖАЛСЯ" по конкретному вопросу — **признаётся недействительным** по соответствующему вопросу.

Бюллетень должен быть передан / направлен по адресу [...] не позднее [DD.MM.YYYY HH:MM].
```

### Шаг 3. Draft уведомление о заочном голосовании

```markdown
# Уведомление о проведении заочного голосования

**Кому:** [участник / акционер]

В соответствии с [ст.38 ФЗ-14 / ст.50 ФЗ-208 / уставом п.X] уведомляем о
проведении внеочередного общего собрания участников / акционеров [организации]
в форме заочного голосования.

**Дата начала:** [DD.MM.YYYY]
**Дата окончания (срок сдачи бюллетеней):** [DD.MM.YYYY HH:MM]
**Способ передачи бюллетеней:** [почта России / курьер / личная доставка по адресу / иной]

## Повестка дня

1. [Вопрос 1]
2. [Вопрос 2]

## Материалы

С материалами по повестке дня можно ознакомиться:
- По адресу [...]  с [время] по [время]
- Электронно по запросу на email [...]

## Прилагаемые документы

1. Бюллетень для голосования
2. [Иные материалы]

**[Должность подписавшего, ФИО]**
**[Подпись] [Дата]**
```

### Шаг 4. Сбор бюллетеней + подсчёт

При получении каждого бюллетеня:
- Проверка соответствия требованиям (подпись, отметки)
- Логирование (журнал входящей корреспонденции / regishtrator log)
- Хранение оригиналов в архиве

После истечения срока:
- Подсчёт голосов по каждому вопросу
- Quorum check (по уставу — обычно 50% + 1 для ООО ст.37; для АО — по ст.58)
- Определение результатов

### Шаг 5. Draft протокол

```markdown
# ПРОТОКОЛ № [N]
# внеочередного общего собрания участников / акционеров
# [Организационно-правовая форма + наименование]
# в форме заочного голосования

**Место подведения итогов:** [адрес]
**Дата начала голосования:** [DD.MM.YYYY]
**Дата окончания (последний день приёма бюллетеней):** [DD.MM.YYYY HH:MM]
**Дата составления протокола:** [DD.MM.YYYY]

## Председательствующий и секретарь

- Председательствующий: ФИО
- Секретарь: ФИО

(Для ООО — обычно председатель общего собрания / директор; для АО — председатель СД / другой по уставу)

| # | ФИО / наименование | Доля в УК / кол-во голосов | Получен ли бюллетень в срок | Действителен ли бюллетень |

| 1 | Иванов И.И. | 30% | Да (DD.MM.YYYY HH:MM) | Да |

- Получено бюллетеней в срок: [N]
- Признано действительными бюллетеней: [N] — % голосов [X%]

## Quorum

[Если отсутствует — собрание признаётся несостоявшимся, решения не приняты]

## Повестка дня и голосование

### Вопрос 1: [формулировка]

| | За | Против | Воздержался | Не проголосовал |
|---|---|--------|-------------|------------------|
| Голосов | X | Y | Z | W |
| % от голосующих | X% | Y% | Z% | W% |

**Решение принято / не принято** (требуемое большинство по уставу / закону: [X])

### Решение: [текст утверждённого решения]

### Вопрос 2: [...]

[То же]

## Удостоверение протокола

[Для ООО — нотариальное удостоверение / альтернативный способ по уставу]
[Для АО — удостоверение регистратором / нотариусом]

Подписи председательствующего и секретаря:
- Председательствующий: ________________ / [ФИО] /
- Секретарь: ________________ / [ФИО] /

**Дата подписания протокола:** [DD.MM.YYYY]
**Нотариально удостоверено:** [нотариус ФИО, реестр N, дата]
**ИЛИ Удостоверено в иной форме:** [подписи всех участников + дата / аудиовидео-запись по протоколу №...]
```

### Шаг 6. Сохранить + uploded в архив

- Оригиналы бюллетеней — хранение [срок по корп. документации правилам]
- Протокол — в корпоративный архив
- Уведомление участников о принятых решениях (для протоколов значимых вопросов)
- Регистрация изменений в ЕГРЮЛ (если требуется — например, смена директора)

## РФ specific gotchas

### Gotcha 1: ст.67.1 ГК — обязательное удостоверение для ООО

Если ООО ОСУ:
- Решение принято — нотариально удостоверено
- **Если не удостоверено** — решение оспоримо (ст.181.4 ГК)
- Альтернативные способы должны быть **в уставе предусмотрены** и **прозрачно применяться**

### Gotcha 2: Для АО — регистратор

Реестродержатель — не просто записывает результаты, **должен присутствовать** и удостоверить процесс. Игнорирование чревато недействительностью.

### Gotcha 3: ст.50 ФЗ-208 запреты

Решение об избрании СД заочно для АО — **automatically void**. Не draft proposal на заочное по этой теме — recommend очное.

### Gotcha 4: Подача в ФНС после изменения

Если результат голосования — изменения в ЕГРЮЛ (смена директора, изменение устава, изменение состава участников ООО):
- Подача формы Р13014 в **7 раб.дней**
- Для ООО — через нотариуса при изменении долей (нотариус сам подаёт)

## Что НЕ делает

- Не подписывает протокол вместо председателя
- Не удостоверяет нотариально (нужен реальный нотариус)
- Не подаёт документы в ФНС
