# /cold-start-interview (corporate-law)

## Назначение

Первый запуск pack. Без заполненного PROFILE.md:
- `/diligence-issue-extraction` не знает thresholds для material flags
- `/closing-checklist` не знает что обязательно нотариус (если ООО)
- `/entity-compliance` не знает корпоративный календарь
- `/board-minutes` не знает структуру органов

## Workflow

### Шаг 1. Тип организации

| Тип | Регулируется | Key features |
|-----|--------------|--------------|
| **ООО** | ФЗ-14 «Об ООО» + ГК | Минимум 1 участник, max 50; УК минимум 10,000 руб; **доли передаются через нотариуса (ст.21)** |
| **АО непубличное** | ФЗ-208 «Об АО» + ГК | Реестр акций ведёт регистратор; нет публичных торгов |
| **ПАО** | ФЗ-208 + ФЗ-39 (рынок цб) + Банк России акты | Раскрытие информации, отчётность Банку России, корп.управление по best practices |
| **НКО (АНО, фонд, ассоциация)** | ФЗ-7 «О НКО» | Цели не предпринимательские; распределение прибыли запрещено |
| **ИП** | ГК + НК | Не юр.лицо; полная ответственность ИП по обязательствам |

→ Ask, store

- Numerodef участников

  - Участник 1: ФИО / название юр.лица + % доли + номинальная стоимость

### Шаг 4. Органы управления

#### Общее собрание

- **ОСУ (для ООО) / ОСА (для АО):**
  - Очередное (годовое) — должно быть проведено по итогам года; срок зависит от типа (ООО — 30 апреля; АО — 30 июня)
  - Внеочередные — по необходимости
  - Кворум по умолчанию (можно изменить уставом):
    - ООО ст.37 — 50% (изменяемо)
    - АО ст.58 — 50% обычных вопросов; повышенный для специальных

#### Совет директоров (Наблюдательный совет)

- **Создан?** Y / N
- Если Y:
  - Численность: [N]
  - Председатель: ФИО
  - Срок полномочий: до следующего годового ОСУ / 1 года / etc.
  - Кворум: [по уставу]
  - Независимые директора: [для ПАО — рекомендуется ≥ 1/3]
- Если N — функции исполняет ОСУ / единственный участник

#### Исполнительный орган

- **Единоличный (ГД, директор, президент):**

  - Срок полномочий: до DD.MM.YYYY
  - Полномочия: устав п.X
- **Коллегиальный (Правление):** Y / N
- **Управляющая компания / управляющий ИП (если применимо):**

### Шаг 5. Material thresholds

#### Крупная сделка (ст.46 ФЗ-14 / ст.78 ФЗ-208)

- Если стоимость отчуждаемого / принимаемого > **25%** балансовой стоимости активов компании по последней отчётности
- → Требует одобрения ОСУ / СД (по уставу)

#### Сделка с заинтересованностью (ст.45 ФЗ-14 / ст.81-84 ФЗ-208)

- Если в сделке участвует interested party (директор, член СД, контролирующий участник, и т.п. — детальная list в законе)
- → Требует одобрения ОСУ (без заинтересованного лица)

#### Internal escalation thresholds

Set by user в PROFILE.md:
- Сделки > [сумма] — escalate к CFO
- Сделки > [сумма] — escalate к CEO
- Сделки > [сумма] / strategic — escalate к Председателю СД

### Шаг 7. Бенефициары (115-ФЗ)

**Обязательно для всех юр.лиц:**
- Кто является бенефициарным владельцем (физ.лицо, имеющее в конечном счёте > 25% или контролирующее)?
- Если "никто" — обоснование (mass ownership, etc.)
- Раскрытие бенефициаров банкам — annually + по запросу
- Раскрытие в ЕГРЮЛ — by reform 2022, операторы должны actualizirovat

### Шаг 8. Корпоративный календарь

- **Ежегодное ОСУ / ОСА:** срок
- **Отчёт ревизионной комиссии (если есть):** срок
- **Бух.отчётность в ФНС:** до 31.03.YYYY
- **Налог на прибыль (декларация):** до 25.03.YYYY
- **Декларация о бенефициарах в ФНС:** ежегодно, до 15.01.YYYY
- **Отчётность регистратору (для АО):** specific даты
- **Раскрытие для ПАО (Банк России):** quarterly, annually + по событиям

### Шаг 9. Risk posture

- **Консервативный:** все corporate actions formal; documentation comprehensive; outside-адв для material
- **Средний:** routine — internal; material — collective + external
- **Агрессивный:** speed > formality. **Note:** не applicable для ПАО (Банк России регулирует)

### Шаг 10. House style / archive

- Шапки документов (template)
- Электронная подпись (КЭП) или собственноручно
- Где хранится корпоративный архив
- Учёт reestre участников / акционеров — самостоятельно или регистратор
- Корпоративный календарь — где ведётся
