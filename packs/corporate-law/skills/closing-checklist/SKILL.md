# /closing-checklist

## Назначение

Generate closing checklist для M&A сделки в РФ — items to be completed for
signing + closing + post-closing.

## Pre-flight

- **Deal structure:**
  - Share deal (доли ООО / акции АО) — основные РФ specifics
  - Asset deal (недвижимость + IP + контракты + персонал)
  - Joint venture (новое юр.лицо)
  - Реорганизация (слияние / присоединение / выделение)

- **Стороны:**
  - Purchaser type (РФ юр.лицо / РФ физ.лицо / иностранец из дружественной страны / иностранец из недружественной — Указ Президента 81/2022!)
  - Seller type (same considerations)
  - Target type (ООО / АО / ПАО)

- **Deal value** (для FAS и валютного контроля thresholds)

## РФ closing checklist — comprehensive

### Phase 1: Signing readiness

#### A. Корпоративные одобрения

| # | Item | Кто | Срок |
|---|------|-----|------|
| 1 | Решение ОСУ/ОСА seller — на отчуждение | Seller | До signing |
| 2 | Решение ОСУ/ОСА buyer — на приобретение (если крупная сделка > 25% активов) | Buyer | До signing |
| 3 | Преимущественное право (для ООО) — offer существующим участникам по ст.21 ФЗ-14 / уставу | Seller | За 30 дней до закрытия |

## РФ-specific gotchas (always check)

### 1. ФАС thresholds (ст.27-28 ФЗ-135)

Согласование ФАС **обязательно** если:
- **Совокупная стоимость активов** group of buyer + target > **800 млн руб** + target > 400 млн, **ИЛИ**
- **Суммарная выручка** group + target > **8 млрд руб** в предыдущем году + target > 400 млн, **ИЛИ**
- Target включён в реестр доминирующих

**Не согласовали** → сделка может быть признана недействительной по иску ФАС; штраф до 500k.

### 2. Указ Президента 81/2022 и последующие

- Если seller — резидент из недружественной страны → разрешение Правкомиссии
- Если buyer — резидент из недружественной → разрешение
- Если структура involves иностранные элементы — обязательная analysis
- **Без разрешения** — сделка не регистрируется (нотариус откажет / регистратор откажет / ФНС откажет в регистрации)

### 3. Преимущественное право (для ООО / непубличного АО)

- **ООО** — другие участники имеют **30 дней** на отказ / acceptance (ст.21 ФЗ-14, если уставом не больше)
- **АО непубличное** — другие акционеры по ст.7 ФЗ-208
- **Не предоставили offer** — другой участник / акционер может требовать перевода прав покупателя на себя в течение **3 мес**

### 4. Согласие супруга

Для физ.лиц-сторон сделок с долями ООО / акциями — **нотариально удостоверённое согласие супруга**, если приобретено в браке (Семейный кодекс ст.35). Иначе сделка оспорима.

### 5. Налоговые consequences

- Для seller: НДФЛ 13% (или 15% при доходе > 5 млн) для физ.лица; налог на прибыль 20% для юр.лица
- **Льгота** — освобождение от НДФЛ при владении долями ООО / акциями > 5 лет (НК ст.217.1)
- Buyer — нет НДС при покупке акций / долей (ст.149 НК)
- Buyer — нет налога на покупку (как US "transfer tax")
- Stamp duty: только госпошлины — небольшие

### 6. Reps & warranties / indemnification

В РФ:
- **ст.431.2 ГК** — заверения об обстоятельствах (с 2015) — основа для R&W
- **ст.406.1 ГК** — возмещение потерь, не связанных с нарушением обязательств — основа для indemnification
- **Без этих ссылок** — R&W / indemnity могут быть оспорены как недействительные

### 7. Эскроу

- **ст.926.1 ГК** — escrow agreement в РФ legalized с 2017
- Banks с лицензией эскроу-агента
- Альтернатива — депозит у нотариуса

### Шаг 3. Risk red flags

#### 🔴 Deal-breakers

- Target в ЕФРСБ (банкротство)
- Уголовное дело против бенефициаров (УК ст.199 / ст.159)
- Sanctioned бенефициары (US OFAC / EU / UK) — заблокирует transfer
- Указ 81/2022 не разрешение, и подходит structure
- Превышение ФАС thresholds без planning согласования

#### 🟡 Material issues — управляемы

- ФАС согласование — timing
- Изменение управляющих органов — формальности
- Bring-down materially adversely changed — recommendation pricing adjustment

## Что НЕ делает

- Не negotiates SPA (это outside-адв)
- Не обращается в ФАС / ЦБ — это outside-адв
- Не подписывает у нотариуса — это сами стороны
