# /entity-compliance

## Назначение

Проверить compliance корпоративного housekeeping за период (annually) или
готовность к event (M&A signing, audit, ФНС проверка).

Output: gap analysis + remediation plan.

## Pre-flight

- Год для review (текущий / прошлый)
- Тип организации (ООО / АО / ПАО / НКО)
- PROFILE.md загружен

## Event-driven обязательства

### При изменении устава / иных учредительных документов

- **Подача формы Р13014** в ФНС в течение **7 раб.дней** с изменения
- Госпошлина: 800 руб
- При повторном — если ошибка / неточность

### При смене единоличного исполнительного органа (директора)

- **Решение** ОСУ / ОСА / СД (по уставу — кто компетентен)
- **Подача формы Р13014** в ФНС в течение **7 раб.дней**
- Уведомление банков (для обновления подписей)
- Внесение в трудовую книжку (если был трудовой договор)

### При изменении состава участников ООО / акционеров АО

#### Для ООО

- **Нотариальное удостоверение сделки** обязательно (ст.21 ФЗ-14) — кроме исключений (внутри-семейные переходы; передача в УК; преобразование)
- Нотариус сам подаёт документы в ФНС в течение **2 раб.дней**
- Преимущественное право других участников — должно быть offer'нуто (ст.21)
- Изменения в уставе если уставом предусмотрено отображение состава участников

#### Для АО

- **Реестродержатель** регистрирует transfer в реестре акционеров — он же информирует ФНС если касается > 25%
- Преимущественное право (для непубличных АО — ст.7 ФЗ-208)
- Уведомления для cross-border сделок (валютный контроль)
- Раскрытие для ПАО (если затрагивает значимых акционеров)

### При увеличении / уменьшении уставного капитала

- Решение ОСУ / ОСА (квалифицированное большинство)
- Изменения в устав
- **Опубликование** в "Вестнике гос.регистрации" — для уменьшения (защита прав кредиторов)
- Подача формы в ФНС
- Внесение в ЕГРЮЛ

### При реорганизации

- Решение ОСУ / ОСА (3/4 голосов — ст.92 ФЗ-208 / ст.51 ФЗ-14)
- **Уведомление кредиторов** в течение 5 раб.дней — публикация в "Вестнике"
- Передаточный акт / разделительный баланс
- Период для кредиторских требований — 2 месяца (ст.60 ГК)
- Госрегистрация — после кредиторского периода

### При ликвидации

- Решение ОСУ / ОСА
- Назначение ликвидационной комиссии
- Уведомление ФНС о ликвидации
- Публикация в "Вестнике гос.регистрации"
- Период для предъявления требований — 2 мес минимум
- Промежуточный + окончательный ликвидационный баланс
- Закрытие банковских счетов
- Подача итоговых форм

## Workflow

### Шаг 1. Annual review checklist

```markdown
# Entity Compliance — [юр.лицо ИНН XXX]
**Год:** [YYYY]
**Дата review:** [DD.MM.YYYY]

## Annual obligations checklist

### 1. Ежегодное общее собрание

- [ ] Проведено: дата [DD.MM.YYYY]
- [ ] Срок соблюдён (до 30.04 / 30.06): Y/N
- [ ] Кворум документально подтверждён
- [ ] Протокол оформлен (по требованиям ст.181.2 ГК — состав информации)
- [ ] Решения нотариально удостоверены (для АО — обязательно ст.67.1 ГК; для ООО — по уставу, по умолчанию обязательно)
- [ ] Бух.отчётность утверждена
- [ ] Распределение прибыли утверждено (или решение о невыплате дивидендов)
- [ ] Аудитор утверждён (если обязателен)

### 2. ФНС отчётность

- [ ] Бух.отчётность за [YYYY] подана до 31.03.[YYYY+1]
- [ ] Декларация по налогу на прибыль за [YYYY] — до 25.03.[YYYY+1]
- [ ] НДС-декларации квартальные — все 4 поданы
- [ ] Декларация о бенефициарах — подана до 15.01.[YYYY]
- [ ] Уведомление о контролируемых сделках (если применимо) — до 20.05.[YYYY]
- [ ] Сведения о среднесписочной численности (форма с 2019 включена в РСВ)

### 3. СФР (с 2023 — единая отчётность)

- [ ] Расчёт по страх.взносам — все 4 квартала + годовой
- [ ] Персонифицированные сведения (ежемесячно — СЗВ-М → с 2023 включён в РСВ)
- [ ] СЗВ-СТАЖ (ежегодно) — до 25.01.[YYYY+1]
- [ ] Сведения об инвалидах / о трудоустройстве (если квоты применимы)

### 4. Росстат

- [ ] Все формы по выборке Росстата сданы
- [ ] Statistical reports

### 5. РКН (152-ФЗ)

- [ ] Уведомление оператора актуально (изменения 2023 — подача расширенной формы)
- [ ] Изменения за год отражены (новые цели, категории ПДн)

### 6. Дополнительно для АО / ПАО

- [ ] Реестр акционеров поддерживается регистратором
- [ ] Раскрытие на сайте (для ПАО — ежеквартальная + годовая отчётность)
- [ ] Сообщения о существенных фактах — все опубликованы вовремя
- [ ] Аффилированные лица — список актуальный
- [ ] Корпоративный секретарь работает (для ПАО)

### 7. ЕГРЮЛ актуальность

- [ ] Текущая выписка получена в [DD.MM.YYYY]
- [ ] Все изменения отражены (директор, адрес, доли, ОКВЭД, и т.д.)
- [ ] Заявления Р13014 поданы по всем изменениям

## Findings

[Issues identified during review]

| # | Issue | Severity | Action | Срок |
|---|-------|----------|--------|------|
| 1 | Годовое ОСУ не проведено в 2025 | 🔴 critical | Provesti ASAP; risk дисквалификации директора | within 30 days |
| 2 | Декларация о бенефициарах за 2024 не подана | 🟡 medium | Подать ASAP; штраф 30-40k (НК ст.119.1) | within 14 days |
| ... | | | | |

## Remediation plan

[Concrete steps]
```

### Шаг 2. Risk classifications

#### 🔴 Critical (могут привести к существенным consequences)

- **Годовое ОСУ не проведено** → дисквалификация директора (КоАП ст.13.25); risk оспаривания всех решений за период
- **Бух.отчётность не сдана / искажена** → штраф до 700k (КоАП ст.15.11)
- **Декларация о бенефициарах не подана** → штраф 30-40k (НК ст.129.1)
- **Контролируемые сделки не задекларированы** → штраф 5000-50000 (НК ст.129.4)
- **ЕГРЮЛ records out of sync** → недостоверность сведений (ст.51 ГК), компания может быть исключена из ЕГРЮЛ

#### 🟡 Medium

- **Срок ОСУ просрочен на < 1 мес** → fixable, штраф per директор
- **Раскрытие для ПАО просрочено** → штраф Банка России до 1М
- **Реестр не updated после изменения** → требует ratification
- **Дочерние компании без своей compliance** → review каждую

#### 🟢 Minor

- Документация некомплектна по форме
- Аудитор еще не назначен (но не критично если можно отнести к следующему году)

### Шаг 3. Event-readiness check (для конкретного event)

Если pre-flight для signing M&A / opening новой credit line / IPO preparation:

#### M&A signing readiness

- [ ] ЕГРЮЛ актуальная выписка (< 30 дней)
- [ ] Реестр участников ООО / акционеров АО актуальный
- [ ] Действующие учредительные документы (текущая редакция устава)
- [ ] Решения о крупных сделках / сделках с заинтересованностью оформлены за всю историю (для materiality)
- [ ] Корп.одобрение текущей сделки прокурировано (если требуется)
- [ ] Бенефициары верифицированы / документированы
- [ ] Иностранный investor — соответствие Указам Президента 2022 (контр-санкции) проверено

#### IPO readiness (для ПАО transformation)

- [ ] Преобразование в ПАО оформлено (ст.7 ФЗ-208)
- [ ] Корпоративные процедуры за 3 года compliant
- [ ] Аудит за 3 года clean
- [ ] Корпоративный кодекс принят (best practice — Кодекс Банка России)
- [ ] Корпоративный секретарь назначен
- [ ] Независимые директора (не менее 1/3)
- [ ] Раскрытие information setup

## Что НЕ делает

- Не подает документы в реестры (это секретарь / outside-адв)
- Не проводит ОСУ (это администрирование)
- Не drafts решения собраний (это `/written-consent` / `/board-minutes`)
