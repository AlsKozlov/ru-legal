## RU-specific quirks (для memo)

- **Нотариус для ООО сделок с долями** — ст.21 ФЗ-14, обязательно (кроме исключений: внутри-семейные передачи; передача компании с одним участником)
- **Реестр акционеров АО** — с 2014 ведётся регистратором, не самим АО (ФЗ-39 ст.8)
- **Преимущественное право** участников ООО / акционеров непубличного АО — обязательно offer first к existing
- **ФАС согласование** — крупные сделки > thresholds (ст.27-28 ФЗ-135) — обязательно
- **СРО / лицензии** — ratings change при смене учредителей
- **Корпоративный договор** — может содержать ограничения опционов, но не отменяет публичной отчётности по бенефициарам (215-ФЗ)
