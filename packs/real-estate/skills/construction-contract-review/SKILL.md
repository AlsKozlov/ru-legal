# /construction-contract-review

## Назначение

Review договоров строительного подряда — от minor remodel до major construction.

## Types

| Type | Применимая norma |
|------|-------------------|
| Generic подряд | ст.702-729 ГК |
| Строительный подряд | ст.740-757 ГК (lex specialis) |
| Подряд на проектно-изыскательские работы | ст.758-762 ГК |
| Бытовой подряд (для физ.лиц) | ст.730-739 ГК + ЗЗПП |
| Подряд для гос.нужд | ст.763-768 ГК + 44-ФЗ |
| ДДУ (для жилого нового строительства) | 214-ФЗ (см. `/ddu-review`) |

## Pre-flight

- **Type** confirmed
- **Side:** заказчик / подрядчик
- **Cost estimate:** [...]
- **Срок:** [...]

## Mandatory elements (ст.740 ГК)

Строительный подряд **должен** содержать:

1. **Конкретный объект** строительства / реконструкции / ремонта
2. **Срок** выполнения
3. **Стоимость** работ
4. **Состав + объём** работ (часто через смету / proектную документацию)

## Workflow

### Шаг 1. Verify mandatory + party rights

```markdown
## Подрядчик due diligence

- [ ] ЕГРЮЛ статус active
- [ ] СРО членство для тех видов работ что требуют:
  - Проектирование (СРО проектировщиков)
  - Строительство (СРО строителей)
  - Изыскания (СРО изыскателей)
- [ ] Опыт relevant (portfolio)
- [ ] Финансовая устойчивость
- [ ] Не в реестре недобросовестных (если гос.заказ применимо)
- [ ] Лицензии (если применимы — некоторые виды работ)

## Заказчик due diligence

- [ ] Подтверждение собственности на объект / земельный участок
- [ ] Разрешение на строительство получено (если требуется — ст.51 ГрК)
- [ ] Проектная документация прошла экспертизу (если obязательна)
```

### Шаг 2. Critical clauses

#### A. Объект + объём работ

```markdown
- [ ] Конкретное описание объекта
- [ ] Адрес / location
- [ ] Кадастровый номер земельного участка
- [ ] Состав работ:
  - Через смету (с детализацией)
  - Через проектную документацию (с reference)
  - Через техническое задание
- [ ] Spec материалов (кто supplies — подрядчик / заказчик)
- [ ] Качество — references к СНиП / ГОСТ / своды правил
- [ ] Дополнительные работы (доп.соглашения требуются?)
```

#### B. Сроки

```markdown
- **Дата начала:** [DD.MM.YYYY]
- **Дата окончания:** [DD.MM.YYYY]
- **Промежуточные сроки** (для multi-stage):
  - Stage 1: дата + что включает
  - Stage 2: дата + что включает
  - ...
- **Условия о продлении сроков** (за форс-мажор / задержки заказчика)
- **Неустойка за просрочку**

⚠ **Red flag** — vague "до закрытия года", "в разумный срок" — недостаточно конкретно
```

#### C. Цена

```markdown
- **Total стоимость:** [сумма]
- **Структура:**
  - Fixed price, ИЛИ
  - "Открытая цена" — по фактически выполненным работам по согласованной смете
- **График оплаты:**
  - Аванс (typical 10-30%) при подписании
  - Промежуточные платежи (по этапам / по %)
  - Окончательный платёж (after acceptance)
- **Удержания** (обычно 5-10% до истечения гарантийного срока — security)
- **Индексация** (для долгосрочных) — formula / CPI

⚠ **Watch:** изменение цены допустимо только в случаях ст.744 ГК (значительное изменение объёма / непредвиденные затраты)
```

#### D. Качество + acceptance

```markdown
- **Standards:**
  - СНиП / своды правил
  - ГОСТы
  - Проектная документация
  - Согласованные образцы

- **Acceptance procedure:**
  - Промежуточная приёмка по этапам
  - Финальная приёмка (акт КС-2)
  - Скрытые работы (специальные акты освидетельствования)

- **Право заказчика отказать** в приёмке при дефектах
- **Срок устранения дефектов** подрядчиком
- **Replacement / refund** если cannot устранить
```

#### E. Гарантийный срок (ст.756 ГК)

```markdown
- **Минимум** для строительства: **5 лет** с даты сдачи объекта
- **Для оборудования:** обычно 2-3 года
- **Условие предъявления претензий**
- **Срок устранения** недостатков подрядчиком
- **Право заказчика** на самостоятельное устранение с возмещением (если подрядчик отказался)

⚠ Если меньше 5 лет — недействительно (для строительства жилых)
```

#### F. Ответственность

```markdown
### Подрядчик liable за:

- Нарушение сроков (неустойка)
- Качество работ (гарантия)
- Сохранность объекта (ст.751)
- Соблюдение СНиП / СП
- Действия субподрядчиков (ст.706)

### Заказчик liable за:

- Задержку оплаты (проценты)
- Задержку предоставления документации / discrepancies
- Непредоставление земельного участка / доступа

### Limitations

- Часто включается max liability cap
- Force majeure clause
- Mutual indemnification
```

#### G. Субподрядчики

```markdown
- [ ] Право подрядчика привлекать суб (default — yes ст.706 ГК)
- [ ] Уведомление заказчика
- [ ] Approval заказчика для key subs
- [ ] Подрядчик remains liable перед заказчиком за всех суб
- [ ] Subs membership СРО (если applicable)
```

#### H. Изменения

```markdown
- [ ] Procedure для изменений объёма работ
- [ ] Дополнительные работы — обязательно в письменной форме
- [ ] Approval prior начала
- [ ] Pricing рrocedure для изменений
- [ ] Срок продления (если applicable)
```

#### I. Расторжение

```markdown
### Right заказчика (ст.717 + 738 ГК)

- В любой момент (с компенсацией) — default rule
- Compensation: фактически выполненные + убытки

### Right подрядчика

- При просрочке оплаты (ст.752)
- При препятствиях со стороны заказчика
- При иных существенных нарушениях

### Notice periods

[...]

### Settlement procedure

[Final payments, returns]
```

#### J. Insurance

```markdown
- [ ] Construction All Risks (CAR) insurance — кто несёт
- [ ] Liability insurance подрядчика
- [ ] Workers compensation (для подрядчиков с employees)
- [ ] Material in transit insurance
```

### Шаг 3. РФ-specific traps

#### Trap 1: Разрешение на строительство

Без разрешения (когда оно требуется — ст.51 ГрК) — самовольная постройка (ст.222 ГК) → может быть снесена + штрафы.

→ Verify до подписания договора.

#### Trap 2: Перепланировка / реконструкция жилья

- Требует согласование с органом местного самоуправления (для жилых)
- Без согласования — административная ответственность (КоАП ст.7.21) + риск sноса
- Регистрация изменений в ЕГРН (через Росреестр)

#### Trap 3: Самовольные изменения в commercial объектах

- Если объект — в составе ОКН (объект культурного наследия) — особые ограничения
- Если в исторической части города — ограничения

#### Trap 4: Иностранный подрядчик

- Лицензирование РФ требуется
- СРО membership — иногда impossible для иностранцев
- Часто работают через РФ дочернюю / agent компанию

## Что НЕ делает

- Не handles разрешения / согласования (это сам заказчик / outside service)
- Не handles технический надзор (стороннее лицо)
- Не replaces outside-адв ФПА для material матерей
