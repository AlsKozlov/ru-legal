# /ddu-review

## Назначение

Killer skill для покупателей жилья на этапе строительства. Review draft ДДУ
перед подписанием — flag нарушения 214-ФЗ + риски + не-required clauses
ущемляющие права участника.

## Pre-flight

- **Застройщик identified:** ИНН + проверка через ЕГРЮЛ (давно работает? есть other projects? в реестре проблемных застройщиков?)
- **Объект:** адрес ЖК + конкретная квартира (номер, площадь, этаж, очередь)
- **Стоимость:** общая сумма + структура (cash / ипотека / маткапитал)
- **Дата планируемого подписания:** [DD.MM.YYYY]

## Mandatory elements ДДУ (ст.4 214-ФЗ)

ДДУ **должен** содержать (без этого — недействителен):

1. **Определение конкретного объекта** долевого строительства (с проектной декларацией)
2. **Срок передачи** объекта застройщиком участнику
3. **Цена** ДДУ + порядок уплаты
4. **Гарантийный срок** для объекта (не менее 5 лет — для constructions, 3 лет — для тех.-нженерного оборудования)
5. **Способы обеспечения** исполнения обязательств застройщиком (эскроу или гарантийный фонд)

## Workflow

### Шаг 1. Background check застройщика

```markdown
## Застройщик due diligence

### Через ЕГРЮЛ

- ИНН / ОГРН actual?
- Учредители stable history?
- Уставный капитал
- Дата основания (vs project старт)

### Через Единый реестр застройщиков (Минстрой / наш.дом.рф)

- Зарегистрирован? Статус?
- Сданные projects history
- Open projects status (по графику или delays?)
- В реестре проблемных? (если да — STOP)

### Через ЕФРСБ (банкротство)

- Не банкрот?
- Аффилированные не банкроты?

### Через kad.arbitr

- Open споры с participants
- Иски от других participants

### Через нашуданные (Единый ресурс жилищного строительства)

- Проектная декларация published
- Финансовая отчётность published
- Изменения вовремя
```

### Шаг 2. Verify проектная декларация

```markdown
## Проектная декларация (ст.19 214-ФЗ)

- [ ] Доступна на наш.дом.рф / сайте застройщика
- [ ] Соответствует фактическому ходу строительства
- [ ] Срок ввода в эксплуатацию указан
- [ ] Спецификации объекта (площадь, этажность, площадь квартир) match ДДУ
```

### Шаг 3. Review specific clauses

#### A. Объект ДДУ

```markdown
- [ ] Конкретно identified (адрес, корпус, секция, этаж, № квартиры)
- [ ] Площадь указана: проектная (могут change ± 5%)
- [ ] Условия о изменении площади (price adjustment)
- [ ] Балконы / лоджии (учитываются в площади с коэффициентами)
- [ ] Парковочное место (если planned — отдельный ДДУ или включено?)
- [ ] Кладовка / нежилые помещения
- [ ] Соответствие проектной декларации
```

#### B. Срок передачи

```markdown
- [ ] Срок передачи (конкретная дата, не "в течение N квартала") — best practice
- [ ] Connections с разрешением на ввод в эксплуатацию
- [ ] Подписание акта приёма-передачи — конкретные процедуры

⚠ **Red flag** — "не позднее [Q4 N+2 года]" — vague, may delay actually.
⚠ **Red flag** — "после получения разрешения на ввод" без firm date — застройщик может затянуть indefinitely.
```

#### C. Цена

```markdown
- [ ] Total стоимость указана конкретно
- [ ] Структура оплаты (единоразово / графиком)
- [ ] Эскроу счёт — обязательно с 01.07.2019 для договоров с участниками (ст.15.4 214-ФЗ)
- [ ] Банк эскроу-агент указан (с лицензией Банка России)
- [ ] Условие о возврате средств с эскроу при non-completion

⚠ **Red flag** — "договор беспроцентного займа" / "инвестиционный договор" вместо ДДУ — обход 214-ФЗ, нет защиты!
⚠ **Red flag** — "напрямую на счёт застройщика" без эскроу — strict нарушение
```

#### D. Гарантийный срок (ст.7 214-ФЗ)

```markdown
- [ ] Гарантия на объект — не менее **5 лет** с даты передачи
- [ ] Гарантия на технологическое + инженерное оборудование — не менее **3 лет**
- [ ] Условия предъявления претензий + сроки устранения недостатков
- [ ] Право замены / возврата стоимости при substantial defects

⚠ **Red flag** — гарантия меньше required — недействительно условие
⚠ **Red flag** — "гарантия не распространяется на [...]" с broad list — limits защиту
```

#### E. Ответственность застройщика

```markdown
## Ст.6 214-ФЗ — ответственность за нарушение срока

При нарушении срока передачи — **неустойка**:

- **Для гражданина:** 1/150 ставки рефинансирования ЦБ от цены договора **за каждый день** просрочки (т.е. ~6% годовых при ставке 7.5%)
- **Для юр.лиц / ИП-инвесторов:** 1/300

→ Это imperative — НЕ может быть reduced договором.

[ ] Условие неустойки соответствует ст.6 или нет (если меньше — недействительно)
[ ] Условие что неустойка покрывает в т.ч. моральный вред (ст.151 ГК + ст.15 ЗЗПП)
```

##### 🚨 Мораторий на неустойки 2022-2024 (CRITICAL для 2024-2025 кейсов!)

**Постановление Правительства РФ № 326 от 18.03.2024** (предшественники — № 479 от 26.03.2022, обновлялось ПП № 442 от 30.09.2022) — установило особенности применения **неустоек, штрафов, процентов и иных финансовых санкций** по ДДУ:

| Период | Действие | Источник |
|--------|----------|----------|
| 29.03.2022 — 30.06.2023 | Неустойки **не начислялись** | ПП № 479 от 26.03.2022 |
| 22.03.2024 — 31.12.2024 | Неустойки **не начисляются** | ПП № 326 от 18.03.2024 |
| 01.01.2025+ | Возможно продление — verify актуальный ПП | check pravo.gov.ru |

**Что это значит на практике:**

- Если просрочка передачи приходится на «мораторный» период — неустойка по 1/150 ставки ЦБ **за эти дни не подлежит начислению**
- Для расчёта неустойки **исключить мораторные периоды** из общего срока просрочки
- При составлении претензии/иска — **указать применимые периоды моратория**, иначе суд снизит сумму с мотивировкой «мораторий»
- Период моратория **не учитывается** при расчёте 2 мес для расторжения по ст.9 (срок «приостанавливается»)
- Под мораторий попадают: неустойка ст.6, штраф 50% за неудовлетворение требований потребителя (ЗЗПП ст.13), проценты ст.395 ГК
- **НЕ попадают:** требования об устранении недостатков (ст.7), компенсация морального вреда, возврат денег по расторжению

⚠ Это **самая частая причина проигрыша покупателей в 2024-2025** — рассчитывают неустойку без учёта моратория, получают отказ или резкое снижение.

#### F. Качество объекта

```markdown
- [ ] Технические характеристики quartиры спецификированы (отделка / без / черновая / чистовая)
- [ ] Материалы конкретно
- [ ] Размер refundу если меньше площадь
- [ ] Возможность отказа от ДДУ если несоответствие условиям
- [ ] Право на устранение недостатков застройщиком в разумный срок

⚠ **Red flag** — "застройщик вправе изменять планировку без согласия" — недопустимо
⚠ **Red flag** — "не несёт ответственности за отделку" если в declaration отделка обещана
```

#### G. Расторжение по требованию участника

```markdown
## Ст.9 214-ФЗ — основания расторжения

Участник может **в одностороннем порядке** расторгнуть если:

- Просрочка передачи > **2 мес** от срока ДДУ
- Существенные нарушения качества (определены ст.7)
- Существенное изменение проекта (площадь, планировка > 5%)
- Прекращение или приостановление строительства давшие основания полагать что объект не будет передан в срок

→ Возврат уплаченных средств + неустойка + проценты за пользование (1/300 ст.рефинансирования)

[ ] Условие в ДДУ не препятствует этим правам?
```

#### H. Регистрация (ст.4 ч.3 214-ФЗ)

```markdown
- ДДУ **должен быть зарегистрирован** в ЕГРН — обязательно
- Только после регистрации — права защищены
- Оплата за регистрацию — обычно participant (но можно negotiate)
- Срок регистрации — typically 5-7 раб.дней

⚠ **Red flag** — обязанность платить полную сумму **до** регистрации (особенно без эскроу!)
```

### Шаг 4. Common ущемляющие clauses

**Часто включают в ДДУ, but могут быть оспорены/недействительны:**

1. **"Споры подсудны суду по месту нахождения застройщика"** — для potребительских споров недействительно (potребитель может выбирать суд по месту жительства — ст.17 ЗЗПП)

2. **"Все изменения и дополнения только по соглашению сторон + застройщик resеrves right unилateрally изменять"** — internally противоречиво

3. **"Передача объекта по акту считается завершённой даже при наличии замечаний участника"** — недействительно

4. **"Гарантийный срок не distribuится на [очень broad list]"** — могут ограничить required minimum

5. **"Застройщик не несёт ответственности за изменения сроков по объективным причинам"** — слишком широко

6. **"Договор может быть расторгнут только в judicial порядке"** — нарушает ст.9 (право одностороннее расторжение)

7. **"Запрет уступать права по ДДУ"** — нарушает ст.11 (участник имеет право уступать пока не подписан акт передачи)

8. **"Денежные средства non-refunde even при нарушении"** — недействительно

9. **"Заказчик принимает риск изменения площади без adjustments цены"** — depending — иногда ущемляет (если уменьшение)

## Special considerations

### Покупка с ипотекой

- Bank checks застройщика отдельно — additional layer защиты
- Bank требует эскроу
- Bank insurance может cover defaults
- Watch для "связанных" страховщиков по требованию банка

### Маткапитал

- Сертификат может быть направлен на оплату ДДУ
- Дополнительные требования к застройщику
- Часть ДДУ должна быть на детей (в собственность)

### Иностранец покупатель

- Restrictions на покупку земельных участков (в погранрайонах — запрет)
- Land под жильем — обычно OK для иностранцев
- Указ 81/2022 для недружественных — verify применимость

### Apartments (апартаменты) vs квартиры

- **Apartments** — нежилые помещения (commercial use по документам), даже если used as housing
- 214-ФЗ часто **не применяется** к апартаментам — другая защита
- Higher tax (commercial real estate tax)
- Нет прописки

→ Flag если предлагают апартаменты под видом жилья!

## Что НЕ делает

- Не negotiates с застройщиком (это покупатель + outside-адв)
- Не handles ипотечный договор (это banking)
- Не handles регистрацию в ЕГРН (это нотариус / застройщик / сам purчasely)

**Правовая основа:**
- 214-ФЗ «Об участии в долевом строительстве»
- ГК ст.554-558 (купля-продажа недвижимости)
- ЗЗПП (consumer protection)
- ЖК РФ
- Закон РФ № 2300-1 «О защите прав потребителей»
