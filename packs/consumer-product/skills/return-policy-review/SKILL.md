# /return-policy-review

## Назначение

Verify return policy compliant с ЗЗПП. Customer-friendly clauses + flag
ущемляющих провисов.

## Workflow

### Шаг 1. Identify тип

| Тип | Применимая norma | Период |
|-----|------------------|--------|
| Некачественный товар | ст.18 ЗЗПП | 2 года минимум (или гарантия) |
| Качественный товар (не подошёл) | ст.25 ЗЗПП | **14 дней** (вкл. день покупки) |
| Дистанционная продажа (web / catalog) | ст.26.1 ЗЗПП | **7 дней** после получения (cooling-off) |
| Отказ от услуги | ст.32 ЗЗПП | в любое время до оказания, с компенсацией |
| Digital products / SaaS | ст.18 + ст.25 (analogous) | Зависит от природы |

### Шаг 2. Compliance check per category

#### A. Некачественный товар (ст.18)

```markdown
**Обязательные права потребителя:**
- Возврат денег (refund)
- Замена на аналогичный
- Замена на товар другой марки с пересчётом
- Снижение цены
- Безвозмездное устранение недостатков (ремонт)
- Компенсация расходов на ремонт у третьих лиц

**Период:** 
- 2 года минимум для большинства товаров
- Дольше — если гарантия / срок службы продлёны

**Кто несёт расходы возврата:**
- Продавец (для крупногабаритных > 5 кг)
- Покупатель (для остальных, но может requesting reimbursement если возврат правомерный)

### Red flags в политике:
- ❌ "Только обмен, no refund" — ст.18 даёт право на refund
- ❌ "Refund только bonus-ами" — недействительно
- ❌ "Возврат за свой счёт всегда" — некорректно
- ❌ "Срок < 2 лет" — не может быть меньше
```

#### B. Качественный товар не подошёл (ст.25)

```markdown
**Условия:**
- 14 дней с момента покупки
- Товар не использовался, сохранены товарный вид + потребительские свойства
- Сохранены пломбы / ярлыки / товарный чек

**Перечень не подлежащих обмену** (Постановление Правительства РФ 55):
- Лекарства / БАДы
- Косметика / парфюмерия
- Бельё / трикотаж
- Ткани / коврики
- Скоропортящаяся еда
- Технически сложные товары (если использовались)
- Книги / газеты
- Ювелирные изделия
- И др.

### Red flags:
- ❌ "Без объяснения причин не принимаем" — потребитель не обязан объяснять
- ❌ "Только в день покупки" — 14 дней guaranteed
- ❌ "Только с чеком" — чек не required (можно через другие доказательства)
```

#### C. Дистанционная продажа (ст.26.1)

```markdown
**Cooling-off period: 7 дней с получения**

Любой товар (кроме индивидуально определённых, изготовленных по заказу) — можно
вернуть **без объяснения причин** в 7-дневный срок.

**Условия:**
- Товар сохранён в товарном виде
- Расходы на возврат — на покупателе (если иначе не предусмотрено)

### Red flags:
- ❌ "Дистанционная продажа без cooling-off" — нарушение ст.26.1
- ❌ "Cooling-off только 3 дня" — недостаточно
- ❌ "Cooling-off только для определённых категорий" — applies к всему
```

#### D. Отказ от услуги (ст.32)

```markdown
Потребитель **в любое время** может отказаться от услуги.

**Условия:**
- Оплата фактически понесённых расходов исполнителем

### Red flags:
- ❌ "Услуга предоплачена — refund не делаем" — ст.32 даёт право на refund minus actual costs
- ❌ "Отказ только за 14 дней до" — нельзя ограничить
```

### Шаг 3. Sectoral overlays

#### SaaS

Для SaaS — субscription:
- Cooling-off 7 дней с первого payment (ст.26.1 аналогия)
- Pro-rata refund возможно (best practice)
- Auto-renewal — должно быть informed consent (ст.32 ЗЗПП + ст.426 ГК)

#### Финансовые

- Кредит: 14 дней без объяснения причин (ст.11 ФЗ-353)
- Страхование: 14 дней «период охлаждения» (ст.958 ГК + Указание Банка России)
- ОСАГО: special rules (ст.10 ФЗ-40)
